0. 全文符号约定

为避免不同课件、字幕和 OCR 中符号混乱，本稿统一使用以下记号。

0.1 监督学习基本符号

训练数据记为

[
D_n={(x_i,y_i)}_{i=1}^n,
]

其中 (x_i\in\mathbb R^p) 是特征、协变量、输入变量（features / covariates / predictors），(y_i) 是响应变量、输出变量或结果变量（response / outcome）。

回归模型常写作

[
Y=f(X)+\varepsilon,\qquad E(\varepsilon)=0,\qquad \operatorname{Var}(\varepsilon)=\sigma^2.
]

学习目标是用训练数据构造估计函数 (\hat f)，并在新样本上做预测。需要注意：统计学习关心的是测试误差（testing error），不是单纯的训练误差（training error）。

0.2 分类问题符号

二分类中通常使用

[
y_i\in{0,1}
]

或

[
y_i\in{-1,+1}.
]

逻辑回归（Logistic Regression）一般使用 (0/1) 标签；支持向量机（Support Vector Machine, SVM）通常使用 (-1/+1) 标签。

0.3 因果推断符号

因果推断中观测数据统一写为

[
O_i=(Y_i,A_i,L_i),
]

其中 (Y_i) 是结果变量（outcome），(A_i\in{0,1}) 是处理变量（treatment / exposure），(L_i) 是基线协变量或混杂变量（baseline covariates / confounders）。

潜在结果（potential outcome / counterfactual outcome）记为

[
Y^a,
]

表示“如果人为令处理 (A=a)，将会观察到的结果”。平均处理效应（Average Treatment Effect, ATE）记为

[
\psi=E(Y^1)-E(Y^0).
]

G-公式（G-formula）中记

[
g(a)=E(Y^a),
]

结果回归（outcome regression）记为

[
b(a,L)=E(Y\mid A=a,L),
]

处理机制或倾向得分（treatment mechanism / propensity score）记为

[
f(a\mid L)=P(A=a\mid L).
]

于是

[
g(a)=E{b(a,L)}.
]

0.4 工具变量符号

工具变量（Instrumental Variable, IV）记为 (Z)，处理变量记为 (A)，结果变量记为 (Y)。工具变量下的潜在处理状态记为

[
A^z,
]

即“如果令工具变量 (Z=z)，个体是否会接受处理”。双重干预下的潜在结果可写为

[
Y^{z,a}.
]

排他性限制（exclusion restriction）表示为

[
Y^{z,a}=Y^a.
]

有的 PPT 会写成 (Y_{z,a}=Y_a)，含义相同：固定处理 (A=a) 后，(Z) 不能再直接影响 (Y)。

四类服从类型（compliance types）由 ((A^0,A^1)) 决定：

类型 定义 含义
永不接受者（never takers） (A^0=0,A^1=0) 无论 (Z=0) 还是 (Z=1)，都不接受处理
总是接受者（always takers） (A^0=1,A^1=1) 无论 (Z=0) 还是 (Z=1)，都接受处理
顺从者（compliers） (A^0=0,A^1=1) (Z=1) 时接受，(Z=0) 时不接受
反抗者（defiers） (A^0=1,A^1=0) (Z=1) 时不接受，(Z=0) 时反而接受

1. 总体学习地图

截至 2026-04-24，本课程的主线可以概括为以下几部分。

  1. 监督学习基本问题：给定训练数据 (D_n={(x_i,y_i)}_{i=1}^n)，学习函数 (\hat f)，用于预测新样本。
  2. 回归与预测误差：从 K 近邻（K-Nearest Neighbor, KNN）、线性回归（Linear Regression）、训练误差与测试误差进入偏差-方差权衡（Bias-Variance Trade-off）。
  3. 正则化与变量选择：岭回归（Ridge Regression）、套索回归（Lasso）、弹性网（Elastic Net）、SCAD、MCP。
  4. 分类：逻辑回归（Logistic Regression）、分类评价指标（classification metrics）、线性判别分析（Linear Discriminant Analysis, LDA）与二次判别分析（Quadratic Discriminant Analysis, QDA）。
  5. 间隔分类与核方法：支持向量机（Support Vector Machine, SVM）、核技巧（Kernel Trick）、再生核希尔伯特空间（Reproducing Kernel Hilbert Space, RKHS）、表示定理（Representer Theorem）、核岭回归（Kernel Ridge Regression, KRR）。
  6. 非参数局部方法：核密度估计（Kernel Density Estimation, KDE）、核回归（Kernel Regression）、局部线性回归（Local Linear Regression）。
  7. 树模型与集成学习：分类回归树（Classification and Regression Trees, CART）、剪枝（Pruning）、装袋法（Bagging）、随机森林（Random Forest）、提升法（Boosting）。
  8. 因果推断：潜在结果（Potential Outcomes）、平均处理效应（ATE）、G-公式、逆概率处理加权（Inverse Probability of Treatment Weighting, IPTW）、双重稳健估计（Doubly Robust Estimation, DR）、因果图（DAG）。
  9. 工具变量：工具变量假设、四类服从类型、单调性（Monotonicity）、顺从者平均因果效应（Complier Average Causal Effect, CACE）与 Wald 比值（Wald ratio）。
  10. 计算机视觉：卷积神经网络（Convolutional Neural Network, CNN）、目标检测（Object Detection）、图像分割（Segmentation）、R-CNN 系列、多模态模型（Multimodal Models）。

第 1 讲：KNN、线性回归与偏差-方差权衡

对应课件：KNN_slides.pdf、linearreg_slides.pdf、penalized0_slides.pdf。

1.1 本讲定位

本讲是统计学习课程的入口，核心目的不是只学习某个具体算法，而是建立后续所有模型都会反复使用的三组思想：

  1. 非参数学习与局部平均：以 K 近邻（K-Nearest Neighbor, KNN）为例。
  2. 参数模型与最小二乘：以线性回归（Linear Regression）为例。
  3. 模型复杂度控制：以岭回归（Ridge Regression）和偏差-方差权衡（Bias-Variance Trade-off）为核心。

课堂强调，统计学习不是只追求训练集上拟合得好，而是追求对新样本的预测能力。因此训练误差和测试误差的区别非常重要。

1.2 统计学习基本设定

设训练数据为

[
D_n={(x_i,y_i)}_{i=1}^n.
]

回归问题中常假设

[
Y=f(X)+\varepsilon,
\qquad E(\varepsilon)=0,
\qquad \operatorname{Var}(\varepsilon)=\sigma^2.
]

真实函数 (f) 未知，我们根据训练数据构造估计函数 (\hat f)。在目标点 (x_0) 处，预测值为 (\hat f(x_0))。

重要的是，(\hat f) 不是为了记住训练数据，而是为了在新的测试样本 ((X_0,Y_0)) 上表现好。

1.3 K 近邻回归（KNN Regression）

K 近邻回归的预测规则是

[
\hat f(x_0)=\frac1k\sum_{x_i\in N_k(x_0)}y_i,
]

其中 (N_k(x_0)) 表示训练集中距离 (x_0) 最近的 (k) 个样本点。

这个方法的基本直觉是：离 (x_0) 近的样本更能代表 (x_0) 附近的函数值。它没有假设 (f) 必须是线性函数或某个固定有限维参数族，所以称为非参数方法（nonparametric method）。

KNN 的关键调参量是 (k)。

  • 当 (k=1) 时，预测只看最近的一个点，模型非常灵活，训练误差可能很低，但对噪声非常敏感，方差较大。
  • 当 (k) 很大时，预测由很多点平均得到，函数更平滑，方差较小，但可能忽略局部结构，偏差较大。
  • 当 (k=n) 时，预测退化为全体样本均值，模型极其简单。

所以 KNN 中 (k) 越大，模型越平滑，复杂度越低；(k) 越小，模型越灵活，复杂度越高。

1.4 偏差-方差权衡（Bias-Variance Trade-off）

对于平方损失，在固定目标点 (x_0) 处，预测误差可以分解为三部分：

[
E{(Y_0-\hat f(x_0))^2}
=\sigma^2+igl(E\hat f(x_0)-f(x_0)\bigr)^2+
\operatorname{Var}{\hat f(x_0)}.
]

其中：

  • (\sigma^2) 是不可约误差（irreducible error），来自随机噪声 (\varepsilon)，无法通过改进模型完全消除。
  • (\bigl(E\hat f(x_0)-f(x_0)\bigr)^2) 是偏差平方（squared bias），衡量模型平均预测和真实函数之间的系统偏离。
  • (\operatorname{Var}{\hat f(x_0)}) 是方差（variance），衡量训练数据变化时估计函数的波动程度。

复杂模型通常偏差小但方差大；简单模型通常方差小但偏差大。统计学习的核心任务之一，就是在偏差和方差之间找到合适平衡。

对 KNN：

  • (k) 小：模型复杂度高，偏差小，方差大，容易过拟合（overfitting）。
  • (k) 大：模型复杂度低，偏差大，方差小，容易欠拟合（underfitting）。

1.5 模型复杂度（Model Complexity）

模型复杂度描述模型可以拟合多复杂函数的能力。在线性回归中，模型复杂度通常与变量个数或自由度（degrees of freedom）相关；在 KNN 中，模型复杂度由 (k) 控制。

对 KNN：

[
\text{复杂度}\uparrow \quad \Longleftrightarrow \quad k\downarrow.
]

课件中还提到自由度的一般定义：

[
\operatorname{df}(\hat f)=\frac1{\sigma^2}\sum_{i=1}^n \operatorname{Cov}(\hat y_i,y_i).
]

对于 KNN，近似有

[
\operatorname{df}\approx \frac nk.
]

所以 (k) 越小，自由度越大，模型越复杂。

1.6 训练误差与测试误差

训练误差（training error）定义为

[
\frac1n\sum_{i=1}^n (y_i-\hat f(x_i))^2.
]

测试误差（testing error）是新样本上的期望损失。随着模型复杂度增加，训练误差通常下降，但测试误差常呈 U 型：先下降后上升。

因此，不能只用训练误差选择模型。复杂度太低会欠拟合，复杂度太高会过拟合。

常用模型选择方法包括：

  • 交叉验证（Cross-Validation）
  • 验证集（validation set）
  • 信息准则（AIC/BIC）
  • 正则化路径选择

1.7 交叉验证（Cross-Validation）

K 折交叉验证（K-fold Cross-Validation）步骤如下：

  1. 将数据随机分成 (K) 份。
  2. 每次用 (K-1) 份训练模型，用剩下一份计算测试误差。
  3. 轮流把每一份作为测试集。
  4. 最后把 (K) 次测试误差取平均。

交叉验证用于选择调参量，例如 KNN 中的 (k)，岭回归或 Lasso 中的 (\lambda)。

留一交叉验证（Leave-One-Out Cross-Validation, LOOCV）是 (K=n) 的特殊情况，较稳定但计算成本较高。

1.8 线性回归（Linear Regression）

线性回归假设

[
f(x)=x^\top\beta.
]

矩阵形式为

[
y=X\beta+\varepsilon.
]

普通最小二乘（Ordinary Least Squares, OLS）估计为

[
\hat\beta_{OLS}=\arg\min_\beta |y-X\beta|_2^2.
]

在 (X^\top X) 可逆时，闭式解为

[
\hat\beta_{OLS}=(X^\top X)^{-1}X^\top y.
]

OLS 的优点是简单、可解释、计算方便；缺点是在变量高度相关、维度较高或噪声较大时，估计方差可能很大。OLS 无偏不意味着预测误差最小，因为适当引入偏差可能显著降低方差。

1.9 岭回归（Ridge Regression）

岭回归目标函数为

[
\hat\beta_{ridge}
=\arg\min_\beta{|y-X\beta|_2^2+\lambda|\beta|_2^2}.
]

求导得到

[
\hat\beta_{ridge}=(X^\top X+\lambda I)^{-1}X^\top y.
]

如果目标函数写成

[
\frac1n|y-X\beta|_2^2+\lambda|\beta|_2^2,
]

则解中对应出现 (n\lambda I)。不同课件可能常数不同，但含义相同。

岭回归的作用：

  1. 缩小系数，降低方差。
  2. 缓解多重共线性。
  3. 当 (X^\top X) 不可逆时，(X^\top X+\lambda I) 通常可逆。

岭回归的限制是：它通常不会把系数压成 0，因此不能直接做变量选择（variable selection）。

第 2 讲：惩罚回归、Lasso 与变量选择

对应课件：penalized_slides.pdf、penalized0_slides.pdf。

2.1 本讲定位

本讲从岭回归过渡到 Lasso 和一般惩罚回归（penalized regression）。核心问题是：在变量较多、高维或变量相关性强时，如何同时实现预测稳定和模型可解释？

岭回归能稳定预测，但不能变量选择；Lasso 使用 (L_1) 惩罚，可以同时收缩系数和产生稀疏解。

2.2 惩罚回归统一框架

一般惩罚回归可写为

[
\hat\beta
=\arg\min_\beta
\left{\frac1{2n}|y-X\beta|2^2+
\sum{j=1}^p P_\lambda(\beta_j)\right}.
]

其中 (P_\lambda) 是惩罚函数（penalty function），(\lambda) 是调参量（tuning parameter）。

常见惩罚包括：

方法 惩罚项 特点
岭回归（Ridge） (\lambda\sum_j\beta_j^2) 收缩系数，通常不产生 0
Lasso (\lambda\sum_j \beta_j
最优子集（Best Subset） (\lambda\sum_j1{\beta_j\ne0}) 直接控制非零变量个数，但计算困难
弹性网（Elastic Net） (\lambda_1|\beta|_1+\lambda_2|\beta|_2^2) 兼顾稀疏性与稳定性
SCAD / MCP 非凸惩罚 减少 Lasso 对大系数的偏差

2.3 Lasso 的目标函数

Lasso（Least Absolute Shrinkage and Selection Operator）的目标函数为

[
\hat\beta_{lasso}
=\arg\min_\beta
\left{\frac1{2n}|y-X\beta|_2^2+\lambda|\beta|_1\right}.
]

其中

[
|\beta|1=\sum{j=1}^p|\beta_j|.
]

约束形式为

[
\min_\beta |y-X\beta|_2^2
\quad \text{s.t.}\quad |\beta|_1\le t.
]

Lasso 能产生稀疏解的原因有两种解释。

2.3.1 几何解释

岭回归的 (L_2) 约束区域是圆或球，边界光滑，最优解通常不会正好落在坐标轴上，因此系数通常不为 0。

Lasso 的 (L_1) 约束区域是菱形或多面体，有尖角，尖角位于坐标轴上。损失函数等高线与约束区域相切时，很容易切在尖角处，从而使某些 (\beta_j=0)。

2.3.2 软阈值解释

在正交设计下，Lasso 解具有显式软阈值形式。设 (X^\top X=I)，OLS 对应坐标为 (z_j)。若目标函数写为

[
\frac12(\beta_j-z_j)^2+\lambda |\beta_j|,
]

则

[
\hat\beta_j
=\operatorname{sign}(z_j)(|z_j|-\lambda)_+.
]

也就是

[
\hat\beta_j=
\begin{cases}
z_j-\lambda, & z_j>\lambda,\
0, & |z_j|\le \lambda,\
z_j+\lambda, & z_j<-\lambda.
\end{cases}
]

如果目标函数写为

[
(\beta_j-z_j)^2+\lambda |\beta_j|,
]

则阈值变为 (\lambda/2)：

[
\hat\beta_j=
\begin{cases}
z_j-\lambda/2, & z_j>\lambda/2,\
0, & |z_j|\le \lambda/2,\
z_j+\lambda/2, & z_j<-\lambda/2.
\end{cases}
]

所以考试时一定要看清目标函数平方项前面的常数。回忆卷中出现 (\lambda/2)，正是因为平方项没有写 (1/2)。

2.4 Ridge、Lasso 与 Elastic Net 比较

岭回归（Ridge Regression）：

  • 惩罚为 (L_2) 范数。
  • 能降低方差，缓解多重共线性。
  • 通常不产生稀疏解，不能直接变量选择。

Lasso：

  • 惩罚为 (L_1) 范数。
  • 能产生稀疏解，实现变量选择。
  • 对大系数也会持续收缩，因此估计有偏。
  • 在强相关变量中可能选择不稳定。

弹性网（Elastic Net）：

[
\lambda_1|\beta|_1+\lambda_2|\beta|_2^2.
]

它结合了 Lasso 的稀疏性和 Ridge 的稳定性。对于一组高度相关变量，Elastic Net 往往更倾向于一起保留，而不是像 Lasso 那样只选其中一个。

2.5 SCAD 与 MCP

Lasso 的缺点是对所有非零系数都施加线性惩罚，大系数也会被明显压缩。SCAD（Smoothly Clipped Absolute Deviation）和 MCP（Minimax Concave Penalty）是非凸惩罚，设计目标是：

  1. 对小系数施加强惩罚，使其变成 0。
  2. 对大系数施加较弱惩罚，减少估计偏差。

这类方法理论上可能具有 oracle property，但目标函数非凸，计算更复杂，也可能存在局部最优问题。

2.6 Group Lasso 与 Fused Lasso

当变量具有结构时，可以使用结构化惩罚。

组 Lasso（Group Lasso）适合变量天然成组的情况，以组为单位选择变量。

融合 Lasso（Fused Lasso）鼓励相邻系数相等或相近，适用于有序变量、时间序列或空间结构。

第 3 讲：分类、逻辑回归、LDA/QDA 与 SVM

对应课件：classification_slides.pdf、SVM_slides.pdf。

3.1 分类问题设定

分类问题中，训练数据为

[
D_n={(x_i,y_i)}_{i=1}^n.
]

二分类时，(y_i\in{0,1}) 或 (y_i\in{-1,+1})。分类器（classifier）是一个函数

[
C:\mathbb R^p\to {0,1}.
]

常用损失函数是 0-1 损失（0-1 loss）：

[
L(C(x),y)=1{C(x)\ne y}.
]

理论最优分类器是贝叶斯分类器（Bayes classifier）。对于 (0/1) 二分类，令

[
\eta(x)=P(Y=1\mid X=x),
]

则

[
C^*(x)=1{\eta(x)>1/2}.
]

3.2 软分类与硬分类

软分类器（soft classifier）先估计条件概率 (P(Y\mid X))，再根据阈值分类。逻辑回归就是典型例子。

硬分类器（hard classifier）直接估计分类边界，不一定显式给出概率。SVM 是典型例子。

很多方法可以扩展为二者兼具，但核心思路不同。

3.3 逻辑回归（Logistic Regression）

逻辑回归直接建模

[
\eta(x)=P(Y=1\mid X=x).
]

因为概率必须在 ([0,1]) 中，所以用 logit 链接函数（logit link）：

[
\log\frac{\eta(x)}{1-\eta(x)}=x^\top\beta.
]

等价地，

[
\eta(x)=\frac{\exp(x^\top\beta)}{1+\exp(x^\top\beta)}.
]

3.3.1 系数解释

(\beta_j) 不是 (x_j) 增加 1 时概率增加多少。正确解释是：当 (x_j) 增加 1 且其他变量不变时，log odds 增加 (\beta_j)。

odds 为

[
\frac{P(Y=1\mid X=x)}{P(Y=0\mid X=x)}.
]

因此 (x_j) 增加 1 时，odds 乘以 (\exp(\beta_j))。

3.3.2 极大似然估计

若 (Y_i\mid X_i=x_i\sim \operatorname{Bernoulli}(\eta_i))，其中

[
\eta_i=\frac{\exp(x_i^\top\beta)}{1+\exp(x_i^\top\beta)},
]

则似然函数为

[
L(\beta)=\prod_{i=1}^n \eta_i^{y_i}(1-\eta_i)^{1-y_i}.
]

对数似然为

[
\ell(\beta)=\sum_{i=1}^n\left[y_i x_i^\top\beta-\log{1+\exp(x_i^\top\beta)}\right].
]

逻辑回归通常没有闭式解，需要用牛顿法（Newton-Raphson）或迭代重加权最小二乘（Iteratively Reweighted Least Squares, IRLS）求解。

3.4 分类评价指标

混淆矩阵（confusion matrix）：

预测阳性 预测阴性
实际阳性 TP FN
实际阴性 FP TN

准确率（accuracy）：

[
\operatorname{Accuracy}=\frac{TP+TN}{TP+FP+FN+TN}.
]

灵敏度（sensitivity / recall / true positive rate）：

[
\operatorname{Sensitivity}=\frac{TP}{TP+FN}.
]

特异度（specificity / true negative rate）：

[
\operatorname{Specificity}=\frac{TN}{TN+FP}.
]

假阳性率（false positive rate, FPR）：

[
\operatorname{FPR}=1-\operatorname{Specificity}=\frac{FP}{FP+TN}.
]

精确率（precision）：

[
\operatorname{Precision}=\frac{TP}{TP+FP}.
]

当类别不平衡时，准确率可能误导。例如阴性样本占绝大多数时，永远预测阴性也可能有很高准确率，但灵敏度为 0。

ROC 曲线（Receiver Operating Characteristic Curve）以 FPR 为横轴、TPR 为纵轴，通过改变分类阈值得到。AUC（Area Under the Curve）是 ROC 曲线下方面积，衡量模型排序能力。

3.5 判别分析：LDA 与 QDA

逻辑回归直接建模 (P(Y\mid X))。判别分析（Discriminant Analysis）则先建模每类中的特征分布 (P(X\mid Y=k))，再用贝叶斯公式得到后验概率。

贝叶斯公式给出

[
P(Y=k\mid X=x)=\frac{\pi_k f_k(x)}{\sum_{\ell=1}^K \pi_\ell f_\ell(x)},
]

其中

  • (\pi_k=P(Y=k)) 是类别先验概率；
  • (f_k(x)=P(X=x\mid Y=k)) 是第 (k) 类条件密度。

最优分类规则为

[
\hat y=\arg\max_k \pi_k f_k(x).
]

3.5.1 线性判别分析（LDA）

LDA 假设

[
X\mid Y=k\sim N(\mu_k,\Sigma),
]

即不同类别均值不同，但共享同一个协方差矩阵 (\Sigma)。

判别函数为

[
\delta_k(x)=x^\top\Sigma^{-1}\mu_k- rac12\mu_k^\top\Sigma^{-1}\mu_k+\log\pi_k.
]

分类规则为

[
\hat y=\arg\max_k \delta_k(x).
]

因为 (\delta_k(x)) 是 (x) 的线性函数，所以 LDA 的决策边界是线性的。

3.5.2 二次判别分析（QDA）

QDA 假设

[
X\mid Y=k\sim N(\mu_k,\Sigma_k),
]

允许每个类别有自己的协方差矩阵。判别函数为

[
\delta_k(x)
=-\frac12\log|\Sigma_k|-
\frac12(x-\mu_k)^\top\Sigma_k^{-1}(x-\mu_k)+\log\pi_k.
]

因为含有二次型，所以 QDA 的决策边界通常是二次曲面。

LDA 与 QDA 比较：

  • LDA 参数少，偏差较大，方差较小。
  • QDA 参数多，偏差较小，方差较大。
  • 若各类协方差近似相同，LDA 更稳定。
  • 若样本量充足且协方差明显不同，QDA 更灵活。

3.6 马氏距离分类

如果类别先验相同，且共享协方差矩阵 (\Sigma)，LDA 判别规则等价于选择马氏距离（Mahalanobis distance）最小的类别：

[
d_k^2=(x-\mu_k)^\top\Sigma^{-1}(x-\mu_k).
]

分类规则为

[
\hat C(x)=\arg\min_k d_k^2.
]

考试中如果给定样本点 (x)、几个中心 (\mu_k) 和协方差矩阵 (\Sigma)，只需先求 (\Sigma^{-1})，再分别计算三个二次型，最后选最小者。

3.7 支持向量机（SVM）：线性可分情形

SVM 通常使用标签 (y_i\in{-1,+1})。线性分类函数为

[
f(x)=x^\top\beta+\beta_0.
]

分类规则为

[
\hat y=\operatorname{sign}{f(x)}.
]

线性可分时，希望找到能最大化间隔（margin）的超平面。通过规范化，使支持向量满足函数间隔为 1，可写出硬间隔 SVM（hard-margin SVM）：

[
\min_{\beta,\beta_0}\frac12|\beta|^2
]

subject to

[
y_i(x_i^\top\beta+\beta_0)\ge1,
\qquad i=1,\ldots,n.
]

最大化几何间隔等价于最小化 (|\beta|^2/2)。

3.8 SVM 对偶问题与支持向量

构造拉格朗日函数

[
\mathcal L(\beta,\beta_0,\alpha)
=\frac12|\beta|^2
-\sum_{i=1}^n\alpha_i{y_i(x_i^\top\beta+\beta_0)-1},
]

其中 (\alpha_i\ge0)。一阶条件给出

[
\beta=\sum_{i=1}^n\alpha_i y_i x_i,
]

[
\sum_{i=1}^n\alpha_i y_i=0.
]

代回得到对偶问题

[
\max_\alpha
\sum_{i=1}^n\alpha_i
-\frac12\sum_{i=1}^n\sum_{j=1}^n
\alpha_i\alpha_jy_iy_jx_i^\top x_j
]

subject to

[
\alpha_i\ge0,
\qquad \sum_{i=1}^n\alpha_i y_i=0.
]

互补松弛条件为

[
\alpha_i{y_i f(x_i)-1}=0.
]

因此只有 (\alpha_i>0) 的样本会影响分类超平面，这些样本称为支持向量（support vectors）。

3.9 软间隔 SVM（Soft-Margin SVM）

真实数据常常线性不可分，引入松弛变量（slack variables）(\xi_i)：

[
y_i(x_i^\top\beta+\beta_0)\ge1-\xi_i,
\qquad \xi_i\ge0.
]

软间隔 SVM 优化问题为

[
\min_{\beta,\beta_0,\xi}
\frac12|\beta|^2+C\sum_{i=1}^n\xi_i.
]

其中 (C) 控制间隔宽度与训练错误之间的权衡。

  • (C) 大：更重视训练错误，不允许太多违反间隔，模型更复杂。
  • (C) 小：更允许违反间隔，margin 更宽，模型更稳定。

对偶问题中约束变为

[
0\le\alpha_i\le C.
]

3.10 合页损失（Hinge Loss）

软间隔 SVM 等价于最小化

[
\sum_{i=1}^n[1-y_i f(x_i)]_+ + \lambda|\beta|^2,
]

其中

[
[a]_+=\max(a,0).
]

若样本满足 (y_i f(x_i)\ge1)，则合页损失为 0。也就是说，分类正确且远离边界的点对目标函数没有贡献，这解释了 SVM 的稀疏性。

3.11 核技巧（Kernel Trick）

SVM 对偶问题中样本只以内积 (x_i^\top x_j) 的形式出现。若希望构造非线性分类边界，可以把 (x) 映射到高维特征空间 (\Phi(x))，并使用内积

[
\langle \Phi(x_i),\Phi(x_j)\rangle.
]

核技巧用核函数直接计算这个内积：

[
K(x_i,x_j)=\langle \Phi(x_i),\Phi(x_j)\rangle.
]

这样无需显式写出高维映射 (\Phi)，即可得到非线性 SVM。

第 4 讲：核方法、RKHS、KRR 与局部平滑

对应课件：rkhs_slides.pdf、kernel_slides.pdf、trees_slides.pdf 开头。

4.1 本讲定位

本讲数学内容较多，核心是理解核方法为何既灵活又可计算。主要问题包括：

  1. 核技巧为什么避免显式高维映射？
  2. 再生核希尔伯特空间（RKHS）是什么？
  3. 再生性质有什么作用？
  4. 表示定理为什么能把无限维优化变成有限维优化？
  5. 核岭回归的解如何推出？

4.2 核函数与核技巧

给定特征映射

[
\Phi:\mathcal X\to \mathcal H,
]

若模型依赖 (\Phi(x)) 的内积，则可以定义核函数

[
K(x,z)=\langle \Phi(x),\Phi(z)\rangle_\mathcal H.
]

常见核函数：

  • 线性核（linear kernel）：(K(x,z)=x^\top z)。
  • 多项式核（polynomial kernel）：(K(x,z)=(1+x^\top z)^d)。
  • 高斯核 / RBF 核（Gaussian / radial basis function kernel）：(K(x,z)=\exp{-\gamma|x-z|^2})。

合法核通常要求对任意样本 (x_1,\ldots,x_n)，核矩阵

[
K_{ij}=K(x_i,x_j)
]

是对称半正定矩阵。

4.3 再生核希尔伯特空间（RKHS）

给定核 (K)，对每个点 (x) 定义函数

[
K_x(\cdot)=K(x,\cdot).
]

考虑这些函数的有限线性组合：

[
\mathcal G=
\left{\sum_{i=1}^m\alpha_iK(x_i,\cdot):m<\infty\right}.
]

定义内积：

[
\langle K_x,K_z\rangle_\mathcal H=K(x,z).
]

对于

[
f=\sum_i\alpha_iK(x_i,\cdot),
\qquad
h=\sum_j\beta_jK(z_j,\cdot),
]

定义

[
\langle f,h\rangle_\mathcal H
=\sum_{i,j}\alpha_i\beta_jK(x_i,z_j).
]

将该空间完备化后得到 RKHS。

4.4 再生性质（Reproducing Property）

RKHS 的核心性质是

[
f(x)=\langle f,K_x\rangle_\mathcal H.
]

若

[
f(\cdot)=\sum_i\alpha_iK(x_i,\cdot),
]

则

[
\langle f,K_x\rangle_\mathcal H
=\sum_i\alpha_i\langle K_{x_i},K_x\rangle_\mathcal H
=\sum_i\alpha_iK(x_i,x)=f(x).
]

所以在 RKHS 中，函数取值可以通过内积表示。这是 RKHS 适合做函数估计和核方法的关键。

4.5 表示定理（Representer Theorem）

考虑优化问题

[
\hat f
=\arg\min_{f\in\mathcal H}
\left[
\mathcal L{y_1,f(x_1),\ldots,y_n,f(x_n)}
+\lambda|f|_\mathcal H^2
\right].
]

表示定理说明，最优解必定形如

[
\hat f(\cdot)=\sum_{i=1}^n\alpha_iK(x_i,\cdot).
]

证明思路如下。对任意 (f\in\mathcal H)，将其分解为

[
f=f_\parallel+h,
]

其中

[
f_\parallel\in \operatorname{span}{K_{x_1},\ldots,K_{x_n}},
]

且 (h) 与该有限维空间正交。对任意训练点 (x_j)，有

[
f(x_j)=\langle f,K_{x_j}\rangle
=\langle f_\parallel,K_{x_j}\rangle+\langle h,K_{x_j}\rangle
=f_\parallel(x_j).
]

所以正交部分 (h) 不影响训练损失。另一方面，由勾股定理：

[
|f|\mathcal H^2
=|f\parallel|\mathcal H^2+|h|\mathcal H^2
\ge |f_\parallel|_\mathcal H^2.
]

因此在最优解中必须有 (h=0)。所以最优函数一定落在由训练样本生成的有限维空间中。

这一定理的意义是：虽然 RKHS 可能是无限维空间，但惩罚经验风险最小化的解只需要 (n) 个系数 (\alpha_i)。

4.6 核岭回归（Kernel Ridge Regression）

核岭回归目标函数为

[
\min_{f\in\mathcal H}
\frac1n\sum_{i=1}^n{y_i-f(x_i)}^2+\lambda|f|_\mathcal H^2.
]

由表示定理，令

[
f(\cdot)=\sum_{j=1}^n\alpha_jK(x_j,\cdot).
]

记核矩阵 (K\in\mathbb R^{n\times n})，其中

[
K_{ij}=K(x_i,x_j).
]

训练点上的预测向量为

[
\hat y=K\alpha.
]

同时

[
|f|_\mathcal H^2=\alpha^\top K\alpha.
]

目标函数化为

[
\min_\alpha
\frac1n|y-K\alpha|_2^2+
\lambda\alpha^\top K\alpha.
]

求导：

[
-\frac2nK(y-K\alpha)+2\lambda K\alpha=0.
]

若可化简，则得到

[
(K+n\lambda I)\alpha=y.
]

所以

[
\hat\alpha=(K+n\lambda I)^{-1}y.
]

新样本 (x_0) 的预测值为

[
\hat f(x_0)=\sum_{i=1}^n\hat\alpha_iK(x_i,x_0).
]

当使用线性核 (K(x,z)=x^\top z) 时，核岭回归与普通岭回归等价，只是一个在原始参数 (\beta) 上求解，一个在对偶系数 (\alpha) 上求解。

4.7 核密度估计（Kernel Density Estimation）

给定独立同分布样本 (X_1,\ldots,X_n\sim f)，核密度估计为

[
\hat f(x)=\frac1{nh}\sum_{i=1}^nK\left(\frac{x-X_i}{h}\right),
]

其中 (K) 是核函数，(h) 是带宽（bandwidth）。

带宽 (h) 控制估计的局部程度：

  • (h) 小：只看非常近的点，估计更抖动，偏差小，方差大。
  • (h) 大：看更宽区域，估计更平滑，偏差大，方差小。

核函数形状也重要，但实践中带宽选择通常比核函数选择更关键。

4.8 核回归（Kernel Regression）

Nadaraya-Watson 核回归估计为

[
\hat m(x)=\frac{\sum_{i=1}^nK_h(x-X_i)y_i}{\sum_{i=1}^nK_h(x-X_i)},
]

其中

[
K_h(u)=\frac1hK\left(\frac u h\right).
]

它本质上是局部加权平均：离 (x) 越近的样本权重越大。

与 KNN 比较：

  • KNN 选择固定数量的邻居，邻居内常等权。
  • 核回归对样本赋予连续变化的权重，权重随距离增大而衰减。

核回归在边界处可能有偏，因为边界附近核函数不对称。局部线性回归（Local Linear Regression）通过在每个目标点附近拟合局部线性模型来减小边界偏差。

局部线性回归在目标点 (x_0) 处求解

[
\min_{\beta_0,\beta_1}
\sum_{i=1}^nK_h(x_i-x_0)
{y_i-\beta_0-\beta_1(x_i-x_0)}^2.
]

这仍是加权最小二乘问题。

第 5 讲：树模型、Bagging、随机森林与因果推断导入

对应课件：trees_slides.pdf、causal_intro_slides.pdf。

5.1 本讲定位

本讲先完成树模型和集成学习，再进入因果推断。两部分思维方式不同：树模型和随机森林主要关注预测准确性；因果推断关注干预效应，不能只看相关性。

5.2 CART 基本思想

分类回归树（Classification and Regression Trees, CART）通过递归切分特征空间构造预测规则。

基本流程：

  1. 从根节点开始。
  2. 选择一个变量和一个切分点，将样本分到左右子节点。
  3. 在每个子节点继续递归切分。
  4. 到达停止条件后形成终端节点（terminal nodes）。

预测方式：

  • 回归树：终端节点内 (y) 的平均值。
  • 分类树：终端节点内的多数类或类别比例。

树模型优点：

  • 非参数，不要求线性或正态。
  • 可解释性强。
  • 能处理非线性和变量交互。
  • 对变量单调变换相对稳健。

树模型缺点：

  • 单棵树方差大。
  • 数据轻微变化可能导致树结构大变。
  • 单棵树预测精度通常不如集成方法。

5.3 分类树的不纯度指标

设节点中第 (k) 类比例为 (\hat p_k)。

Gini 指数（Gini index）：

[
G=1-\sum_{k=1}^K\hat p_k^2.
]

熵（entropy）：

[
H=-\sum_{k=1}^K\hat p_k\log\hat p_k.
]

误分类率（misclassification error）：

[
E=1-\max_k\hat p_k.
]

切分时通常选择使子节点加权不纯度下降最大的切分：

[
\Delta I
=I(parent)-\frac{n_L}{n}I(left)-\frac{n_R}{n}I(right).
]

虽然最终分类错误率很重要，但建树时常用 Gini 或 entropy，因为误分类率对局部切分不够敏感。

5.4 回归树切分准则

回归树用节点内平方误差或方差衡量不纯度。切分目标是使左右节点内残差平方和最小，等价于最大化方差下降：

[
\Delta Var
=Var(parent)-\frac{n_L}{n}Var(left)-\frac{n_R}{n}Var(right).
]

5.5 剪枝（Pruning）

树长得太深容易过拟合。两类方法：

  1. 预剪枝（pre-pruning）：提前设定最大深度、最小节点样本量等停止条件。
  2. 后剪枝（post-pruning）：先长大树，再剪掉不划算的子树。

成本复杂度剪枝（cost-complexity pruning）：

[
R_\alpha(T)=R(T)+\alpha |T|,
]

其中 (R(T)) 是树的拟合误差或不纯度，(|T|) 是叶节点数，(\alpha) 控制复杂度惩罚。(\alpha) 越大，越偏好简单树。

5.6 装袋法（Bagging）

Bagging 是 Bootstrap Aggregating 的缩写。

流程：

  1. 从训练集有放回抽样，生成 (B) 个 bootstrap 样本。
  2. 在每个 bootstrap 样本上训练一棵树。
  3. 回归任务取平均；分类任务进行多数投票。

Bagging 主要降低方差（variance），特别适合单棵树这种低偏差、高方差的基学习器。

一个固定样本在一次 bootstrap 抽样中未被抽到的概率为

[
\left(1-\frac1n\right)^n\to e^{-1}\approx0.368.
]

所以每棵树大约有 36.8% 的样本未被使用，这些样本称为袋外样本（out-of-bag samples, OOB）。OOB 样本可以用于估计测试误差和变量重要性。

5.7 随机森林（Random Forest）

随机森林 = Bagging + 随机特征子集。

与普通 Bagging 树相比，随机森林在每次节点切分时，不从全部 (p) 个变量中寻找最优切分，而是随机抽取 (m) 个变量，只在这 (m) 个变量中寻找最优切分。

这样做的核心目的是降低树与树之间的相关性。多个高度相关的树取平均，方差降低有限；随机特征选择可以让树之间更不相似，从而提高集成效果。

随机森林特点：

  • 回归取平均，分类投票。
  • 能处理非线性和交互。
  • 泛化能力强。
  • 可提供变量重要性。
  • 可解释性不如单棵树。

5.8 Boosting 简介

提升法（Boosting）也是集成学习方法，但与 Bagging / Random Forest 不同。

  • Bagging / Random Forest：多棵树基本并行训练，每棵树相对独立。
  • Boosting：模型顺序训练，后一个模型重点修正前一个模型的错误。

Boosting 通常可以降低偏差，但也更容易受噪声和调参影响。

5.9 因果推断导入

统计学习中的预测问题关心 (Y) 能否被 (X) 预测；因果推断关心如果人为改变处理 (A)，结果 (Y) 会如何变化。

两类因果问题：

  • 原因的结果（effect of a cause）：给定处理或干预，评估其影响。
  • 结果的原因（cause of an effect）：观察到结果后追问原因。

本课程主要讨论“原因的结果”。

5.10 潜在结果框架（Potential Outcomes Framework）

对二元处理 (A\in{0,1})，每个个体都有两个潜在结果：

[
Y^1: \text{若接受处理时的结果},
]

[
Y^0: \text{若不接受处理时的结果}.
]

个体因果效应为

[
Y^1-Y^0.
]

但同一个个体只能实际接受一种处理，因此只能观测到

[
Y=AY^1+(1-A)Y^0.
]

另一个潜在结果永远缺失，这就是因果推断的根本问题。

平均处理效应（ATE）定义为

[
\psi=E(Y^1-Y^0)=E(Y^1)-E(Y^0).
]

6.1 本讲定位

本讲是因果推断主体。需要掌握：

  1. 平均处理效应（ATE）如何定义。
  2. 一致性、无未测量混杂和正性假设。
  3. G-公式与 G-计算。
  4. IPTW 的公式和直觉。
  5. 双重稳健估计的形式与证明思路。
  6. 因果图中的混杂变量、碰撞变量和中介变量。
  7. 工具变量三大假设、四类人群、CACE 与 Wald 比值。

6.2 随机试验中的识别

如果处理 (A) 是随机分配，则

[
(Y^0,Y^1)\perp A.
]

若还满足一致性（consistency）

[
Y=Y^A,
]

则

[
E(Y^1)=E(Y\mid A=1),
]

[
E(Y^0)=E(Y\mid A=0).
]

因此

[
\psi=E(Y^1)-E(Y^0)=E(Y\mid A=1)-E(Y\mid A=0).
]

随机试验中可以直接比较处理组和对照组，是因为随机化保证处理分配与潜在结果独立。

6.3 观察性研究中的三个基本假设

观察性研究中，处理 (A) 通常不是随机的。设 (L) 是足够丰富的协变量。需要以下假设。

6.3.1 一致性（Consistency Assumption, CA）

如果个体实际接受处理 (A=a)，则观察到的结果等于该处理下的潜在结果：

[
Y=Y^A.
]

等价地，若 (A=a)，则 (Y=Y^a)。

6.3.2 无未测量混杂（No Unmeasured Confounding Assumption, NUCA）

给定 (L) 后，处理分配与潜在结果独立：

[
(Y^0,Y^1)\perp A\mid L.
]

直观含义是：我们已经测量了足够多的混杂变量，使得在同一 (L) 层内，处理分配像随机试验一样。

6.3.3 正性（Positivity）

对任何 (f_L(l)>0) 的协变量水平，都有

[
P(A=a\mid L=l)>0.
]

也就是说，在每个需要推断的 (L) 层内，都必须既能观察到接受处理的人，也能观察到不接受处理的人。若某类人永远不可能接受处理，就无法从数据中学习他们接受处理时的反事实结果。

6.4 G-公式（G-formula）

在 CA、NUCA 和 positivity 下，(E(Y^a)) 可以由观测数据识别。

若 (L) 离散：

[
g(a)=E(Y^a)=\sum_l E(Y\mid A=a,L=l)P(L=l).
]

若 (L) 连续：

[
g(a)=E(Y^a)=\int E(Y\mid A=a,L=l)f_L(l),dl.
]

记

[
b(a,l)=E(Y\mid A=a,L=l),
]

则

[
g(a)=E{b(a,L)}.
]

推导如下：

[
\begin{aligned}
E(Y^a)
&=E{E(Y^a\mid L)}\
&=\sum_l E(Y^a\mid L=l)f_L(l)\
&=\sum_l E(Y^a\mid A=a,L=l)f_L(l)\qquad\text{由 NUCA}\
&=\sum_l E(Y\mid A=a,L=l)f_L(l)\qquad\text{由 CA}.
\end{aligned}
]

G-计算（G-computation）的 plug-in 估计为

[
\hat g(a)=\frac1n\sum_{i=1}^n\hat b(a,L_i).
]

ATE 估计为

[
\hat\psi_G=\hat g(1)-\hat g(0)
=\frac1n\sum_{i=1}^n{\hat b(1,L_i)-\hat b(0,L_i)}.
]

G-公式依赖结果回归模型 (b(a,L))。若结果回归错设，估计可能有偏。

6.5 IPTW

逆概率处理加权（Inverse Probability of Treatment Weighting, IPTW）不直接建模 (E(Y\mid A,L))，而是建模处理机制

[
f(a\mid L)=P(A=a\mid L).
]

在 CA、NUCA 和 positivity 下，有

[
E(Y^a)=E\left{\frac{I(A=a)}{f(a\mid L)}Y\right}.
]

证明：

[
\begin{aligned}
E\left{\frac{I(A=a)}{f(a\mid L)}Y\right}
&=E\left{\frac{I(A=a)}{f(a\mid L)}Y^a\right}
\qquad\text{由 CA}\
&=E\left[\frac{Y^a}{f(a\mid L)}E{I(A=a)\mid L,Y^a}\right]\
&=E\left[\frac{Y^a}{f(a\mid L)}f(a\mid L)\right]
\qquad\text{由 NUCA}\
&=E(Y^a).
\end{aligned}
]

样本估计为

[
\hat g_{IPTW}(a)=\frac1n\sum_{i=1}^n
\frac{I(A_i=a)}{\hat f(a\mid L_i)}Y_i.
]

特别地，

[
\widehat{E(Y^1)}=\frac1n\sum_{i=1}^n\frac{A_iY_i}{\hat f(1\mid L_i)},
]

[
\widehat{E(Y^0)}=\frac1n\sum_{i=1}^n\frac{(1-A_i)Y_i}{\hat f(0\mid L_i)}.
]

若记 (e(L)=P(A=1\mid L))，则 (f(1\mid L)=e(L))，(f(0\mid L)=1-e(L))。

IPTW 的直觉是构造伪总体（pseudo-population）。若某人在其 (L) 水平下接受当前处理的概率很小，但他实际接受了，则他代表许多类似但没有接受该处理的人，因此权重大。加权后，处理分配在伪总体中不再被 (L) 混杂。

6.6 倾向得分的平衡性质

倾向得分（propensity score）为

[
e(L)=P(A=1\mid L).
]

若 NUCA 成立，则给定倾向得分也可以平衡混杂变量：

[
(Y^0,Y^1)\perp A\mid e(L).
]

直观上，虽然 (L) 可能是高维的，但 (e(L)) 是一维摘要。在正确建模时，处理组和对照组在同一倾向得分层内具有可比性。

6.7 双重稳健估计（Doubly Robust Estimation）

双重稳健估计结合结果回归 (b(a,L)) 和处理机制 (f(a\mid L))。对单个 (a)，定义

[
\hat g_{DR}(a)
=\frac1n\sum_{i=1}^n
\left[
\frac{I(A_i=a)}{\hat f(a\mid L_i)}{Y_i-\hat b(a,L_i)}
+\hat b(a,L_i)
\right].
]

ATE 的 DR 估计为

[
\hat\psi_{DR}=\hat g_{DR}(1)-\hat g_{DR}(0).
]

双重稳健的含义是：只要处理机制模型 (\hat f(a\mid L)) 或结果回归模型 (\hat b(a,L)) 有一个正确，(\hat g_{DR}(a)) 就能一致估计 (E(Y^a))。它不是要求两个模型都正确。

6.7.1 证明：处理机制模型正确

令 (\tilde f,\tilde b) 表示估计模型的概率极限。考虑总体版本

[
\tilde g_{DR}(a)
=E\left[
\frac{I(A=a)}{\tilde f(a\mid L)}{Y-\tilde b(a,L)}+\tilde b(a,L)
\right].
]

若 (\tilde f(a\mid L)=f(a\mid L))，则

[
\begin{aligned}
&E\left[\frac{I(A=a)}{f(a\mid L)}{Y-\tilde b(a,L)}\mid L\right]\
&=\frac1{f(a\mid L)}E[I(A=a){Y-\tilde b(a,L)}\mid L]\
&=\frac1{f(a\mid L)}f(a\mid L)E[Y-\tilde b(a,L)\mid A=a,L]\
&=b(a,L)-\tilde b(a,L).
\end{aligned}
]

所以

[
E\left[
\frac{I(A=a)}{f(a\mid L)}{Y-\tilde b(a,L)}+\tilde b(a,L)
\mid L\right]=b(a,L).
]

对 (L) 取期望：

[
\tilde g_{DR}(a)=E{b(a,L)}=E(Y^a).
]

6.7.2 证明：结果回归模型正确

若 (\tilde b(a,L)=b(a,L)=E(Y\mid A=a,L))，则

[
E[Y-b(a,L)\mid A=a,L]=0.
]

于是

[
\begin{aligned}
&E\left[\frac{I(A=a)}{\tilde f(a\mid L)}{Y-b(a,L)}\mid L\right]\
&=\frac1{\tilde f(a\mid L)}P(A=a\mid L)E[Y-b(a,L)\mid A=a,L]\
&=0.
\end{aligned}
]

所以

[
\tilde g_{DR}(a)=E{b(a,L)}=E(Y^a).
]

这就是双重稳健性。

6.8 因果图（DAG）与调整原则

有向无环图（Directed Acyclic Graph, DAG）用节点表示变量，用箭头表示直接因果关系。

三类常见结构：

6.8.1 混杂变量（confounder）

[
A\leftarrow L\rightarrow Y.
]

(L) 同时影响处理 (A) 和结果 (Y)，是混杂变量。若要识别 (A) 对 (Y) 的总效应，通常需要调整 (L)。

6.8.2 碰撞变量（collider）

[
A\rightarrow C\leftarrow Y.
]

(C) 是碰撞变量。不调整时路径被阻断；一旦调整或条件化于 (C)，会打开路径，引入虚假关联，产生碰撞偏差（collider bias）。

6.8.3 中介变量（mediator）

[
A\rightarrow M\rightarrow Y.
]

(M) 是中介变量。若目标是估计 (A) 对 (Y) 的总效应，不应调整 (M)，否则会阻断部分因果路径。若目标是直接效应，则需要另行定义。

6.9 辛普森悖论（Simpson’s Paradox）

辛普森悖论说明，总体关联和分层关联方向可能相反。它常常由混杂变量导致。因果推断中不能把总体相关性直接解释为因果效应，必须明确调整集合和识别假设。

6.10 工具变量（Instrumental Variable）

当存在未测量混杂时，普通的 G-公式或 IPTW 可能无法识别因果效应。工具变量方法试图利用一个外生变量 (Z) 来识别处理 (A) 对结果 (Y) 的因果效应。

标准 IV 图可以理解为：

[
Z\rightarrow A\rightarrow Y,
]

同时存在未观测混杂 (U) 影响 (A) 和 (Y)。工具变量 (Z) 需要满足：它影响 (A)，但不直接影响 (Y)，并且与未观测混杂无关。

6.10.1 IV 基本假设

相关性（Relevance）

工具变量必须影响处理：

[
E(A\mid Z=1)\ne E(A\mid Z=0).
]

若 (A) 是二元变量，这等价于

[
P(A=1\mid Z=1)\ne P(A=1\mid Z=0).
]

否则 Wald 比值分母为 0，无法识别。

独立性（Independence）

工具变量像随机分配一样，与潜在结果和潜在处理类型独立：

[
Z\perp (Y^1,Y^0,A^1,A^0).
]

这里 ((Y^1,Y^0,A^1,A^0)) 是一个随机向量，表示个体的全部潜在结果和潜在服从类型。这个假设不是说 (Z\perp A)。事实上 IV 要求 (Z) 和观测到的 (A) 相关。独立性说的是 (Z) 不挑人；相关性说的是 (Z) 会推动人的实际处理状态。

由一致性，观测到的处理满足

[
A=ZA^1+(1-Z)A^0.
]

所以即使 (Z\perp(A^1,A^0))，(Z) 仍然可以与观测到的 (A) 相关。

排他性限制（Exclusion Restriction）

工具变量 (Z) 只能通过 (A) 影响 (Y)，不能直接影响 (Y)：

[
Y^{z,a}=Y^a.
]

这意味着固定处理 (A=a) 后，改变 (Z) 不会改变潜在结果。

单调性（Monotonicity）

不存在反抗者（defiers）：

[
A^1\ge A^0.
]

这意味着 (Z) 对 (A) 的影响方向不能对某些人相反。它可以让一部分人从不处理变成处理，但不能让某些人从处理变成不处理。

正值性（Positivity）

工具变量两种取值都要有正概率：

[
0<P(Z=1)<1.
]

否则无法比较 (Z=1) 和 (Z=0) 两组。

6.10.2 四类服从类型

由 ((A^0,A^1)) 定义：

类型 定义 含义
永不接受者（never takers） (A^0=0,A^1=0) 无论 (Z) 如何都不接受处理
总是接受者（always takers） (A^0=1,A^1=1) 无论 (Z) 如何都接受处理
顺从者（compliers） (A^0=0,A^1=1) (Z=1) 接受，(Z=0) 不接受
反抗者（defiers） (A^0=1,A^1=0) (Z=1) 不接受，(Z=0) 反而接受

单调性假设排除 defiers。

6.10.3 CACE 与 Wald 比值

顺从者平均因果效应（Complier Average Causal Effect, CACE）定义为

[
CACE=E(Y^1-Y^0\mid A^1=1,A^0=0).
]

在标准 IV 假设下，CACE 可由 Wald 比值识别：

[
CACE=
\frac{E(Y\mid Z=1)-E(Y\mid Z=0)}
{E(A\mid Z=1)-E(A\mid Z=0)}.
]

分母解释：

[
\begin{aligned}
E(A\mid Z=1)-E(A\mid Z=0)
&=E(A^1)-E(A^0)\
&=E(A^1-A^0).
\end{aligned}
]

在单调性下，(A^1-A^0) 只能是 0 或 1，并且它等于 1 当且仅当个体是 complier。因此

[
E(A\mid Z=1)-E(A\mid Z=0)=P(A^1>A^0),
]

即顺从者比例。

分子解释：由一致性、独立性和排他性限制，

[
E(Y\mid Z=1)-E(Y\mid Z=0)
=E{(Y^1-Y^0)(A^1-A^0)}.
]

在单调性下，(A^1-A^0) 是 complier 指示变量，所以

[
E{(Y^1-Y^0)(A^1-A^0)}
=P(A^1>A^0)E(Y^1-Y^0\mid A^1>A^0).
]

因此

[
\frac{E(Y\mid Z=1)-E(Y\mid Z=0)}
{E(A\mid Z=1)-E(A\mid Z=0)}
=E(Y^1-Y^0\mid A^1>A^0)=CACE.
]

直观地说，Wald 比值等于“(Z) 对 (Y) 的影响”除以“(Z) 对 (A) 的影响”。(Z) 只会通过那些被 (Z) 改变处理状态的人影响结果，因此识别的是顺从者中的平均因果效应，而不是整个总体的 ACE。

对应课件：数据挖掘--计算机视觉_part1.pdf。

7.1 本讲定位

本讲是计算机视觉（Computer Vision）客座内容。它不是前面统计学习主线的直接数学延伸，但展示了统计学习、机器学习和深度学习在图像和视频数据上的应用。

学习重点是：

  1. 计算机视觉的任务类型。
  2. 卷积神经网络为什么适合图像。
  3. 传统视觉方法与深度学习方法的差异。
  4. 多模态模型的发展方向。

7.2 计算机视觉是什么

计算机视觉的目标是让机器理解图像和视频。与普通表格数据相比，视觉数据有以下特点：

  • 维度高。
  • 空间结构强。
  • 局部模式重要。
  • 标注成本高。
  • 输出结构复杂，可能是类别、框、掩膜、文本描述等。

从统计学习角度看，视觉任务仍然是在学习映射 (x\mapsto y)，只是输入 (x) 是图像或视频，模型更复杂，数据维度更高。

7.3 传统视觉与深度学习

传统视觉系统通常分两步：

  1. 手工特征提取（hand-crafted features），例如边缘、纹理、SIFT、HOG。
  2. 使用统计学习分类器，例如 SVM、随机森林、Boosting。

深度学习之后，特征提取和分类器合并为端到端模型：

[
\text{Image}\rightarrow \text{CNN}\rightarrow \text{Feature}\rightarrow \text{Classifier}.
]

模型自动学习特征，而不是手工设计特征。

7.4 卷积神经网络（CNN）

卷积神经网络适合图像的原因：

  1. 局部连接：图像局部像素高度相关。
  2. 权重共享：同一个视觉模式可能出现在不同位置。
  3. 平移等变性：图像平移后，特征响应也相应平移。
  4. 参数量远少于全连接网络。

典型组件包括：

  • 卷积层（convolutional layer）
  • 激活函数（activation function），如 ReLU
  • 池化层（pooling layer）
  • 全连接层（fully connected layer）
  • 归一化层（normalization layer）

7.5 视觉任务分类

图像分类（Image Classification）：输入整张图，输出类别。

目标检测（Object Detection）：输出图中多个目标的位置和类别，位置通常用边界框（bounding box）表示。

语义分割（Semantic Segmentation）：给每个像素分类，但不区分同一类别的不同实例。

实例分割（Instance Segmentation）：既要像素级分类，又要区分不同对象实例。

图像抠图（Image Matting）：更细致地区分前景和背景，尤其处理边缘半透明区域。

视频理解（Video Understanding）：处理时间维度，理解动作、事件和动态关系。

7.6 生成模型与多模态

视觉生成模型（vision generation）目标不是识别已有图像，而是生成新图像或视频。多模态模型（multimodal models）则把图像、文本、视频、音频等统一建模。

常见任务：

  • 文生图（text-to-image generation）
  • 文生视频（text-to-video generation）
  • 图像描述（image captioning）
  • 视觉问答（visual question answering, VQA）
  • 图文检索（image-text retrieval）

第 8 讲：计算机视觉第二部分

对应课件：计算机视觉_part2.pptx。

8.1 本讲定位

本讲继续计算机视觉，重点是目标检测、图像分割、R-CNN 系列、视觉生成和多模态模型。复习重点是任务定义和评价指标，不需要深入每个网络结构的工程细节。

8.2 视觉任务层级

按照输出结构从简单到复杂：

  1. 图像分类（Image Classification）：整张图一个类别。
  2. 分类与定位（Classification and Localization）：识别一个主要对象并给出位置。
  3. 目标检测（Object Detection）：识别多个对象，每个对象有类别和边界框。
  4. 语义分割（Semantic Segmentation）：每个像素分类，不区分实例。
  5. 实例分割（Instance Segmentation）：每个像素分类，并区分不同实例。

8.3 目标检测评价指标

目标检测同时要求类别正确和位置准确。

交并比（Intersection over Union, IoU）：

[
IoU=\frac{Area(B_{pred}\cap B_{true})}{Area(B_{pred}\cup B_{true})}.
]

若 IoU 大于给定阈值，例如 0.5，并且类别正确，则可以认为检测正确。

准确率/精确率（precision）：

[
Precision=\frac{TP}{TP+FP}.
]

召回率（recall）：

[
Recall=\frac{TP}{TP+FN}.
]

平均精度（Average Precision, AP）是某一类别下 precision-recall 曲线的面积。平均平均精度（mean Average Precision, mAP）是多个类别 AP 的平均。

阈值越高，预测越保守，precision 往往更高，recall 可能更低；阈值越低，预测更多，recall 更高，precision 可能下降。

8.4 R-CNN 系列

R-CNN（Region-based Convolutional Neural Network）基本流程：

  1. 用选择性搜索（Selective Search）生成候选区域（region proposals）。
  2. 对每个候选框裁剪并缩放。
  3. 每个候选框单独通过 CNN 提取特征。
  4. 使用分类器判断类别。
  5. 用边界框回归（bounding box regression）修正位置。

R-CNN 的主要缺点是慢，因为每个候选框都要单独跑 CNN，且训练流程复杂。

Fast R-CNN 改进：整张图先通过 CNN 得到共享特征图，再对候选区域做 ROI pooling，避免重复卷积。

Faster R-CNN 改进：用区域建议网络（Region Proposal Network, RPN）替代 Selective Search，使候选框生成也由网络学习。

基本发展脉络：

[
\text{R-CNN}\rightarrow \text{Fast R-CNN}\rightarrow \text{Faster R-CNN}.
]

速度提升主要来自特征共享和 proposal 生成网络化。

8.5 图像分割（Segmentation）

语义分割（Semantic Segmentation）为每个像素预测类别，但不区分同一类别的不同实例。

实例分割（Instance Segmentation）不仅给出像素类别，还要区分对象实例。

应用包括：

  • 医学影像：器官和病灶分割。
  • 自动驾驶：道路、车辆、行人分割。
  • 遥感：建筑、水体、植被分割。
  • 图像编辑：抠图和背景替换。

检测给出边界框，分割给出像素级掩膜（mask）。mask 信息更精细，但标注和训练成本更高。

8.6 多模态模型（Multimodal Models）

多模态模型把图像、文本、视频等输入统一建模。典型结构包括视觉编码器和语言模型，中间通过跨注意力（cross-attention）连接。

核心思想：

  • 视觉编码器提取图像 token。
  • 语言模型处理文本 token。
  • 跨注意力让文本生成过程能够感知视觉信息。

常见任务包括视觉问答、图像描述、图文检索、文生图、视频理解等。

附录 B：概念辨析

B.1 训练误差与测试误差

训练误差衡量模型在训练数据上的拟合程度；测试误差衡量模型在新样本上的泛化能力。复杂模型可能训练误差很低但测试误差很高，即过拟合。

B.2 参数方法与非参数方法

参数方法假设函数属于固定有限维参数族，例如线性回归。非参数方法不预先固定有限维函数形式，模型复杂度通常随样本量变化，例如 KNN、树、核回归。

B.3 Ridge 与 Lasso

Ridge 用 (L_2) 惩罚，主要降低方差，不做变量选择。Lasso 用 (L_1) 惩罚，能产生稀疏解，但对大系数有偏。

B.4 Logistic Regression 与 LDA/QDA

Logistic Regression 直接建模 (P(Y\mid X))。LDA/QDA 建模 (P(X\mid Y))，再用贝叶斯公式得到后验概率。LDA 假设各类共享协方差，QDA 允许各类协方差不同。

B.5 Bagging、Random Forest 与 Boosting

Bagging 通过 bootstrap 训练多个模型并平均，主要降低方差。Random Forest 在 Bagging 基础上增加随机特征子集，降低树之间相关性。Boosting 顺序训练模型，后一个模型修正前一个模型错误，常用于降低偏差。

B.6 G-公式、IPTW 与 DR

G-公式依赖结果回归模型 (b(a,L))。IPTW 依赖处理机制模型 (f(a\mid L))。DR 结合两者，只要其中一个模型正确，就能一致估计目标参数。

B.7 混杂变量、碰撞变量与中介变量

混杂变量需要调整；碰撞变量不能随便调整；中介变量在估计总效应时不应调整。

B.8 工具变量独立性与相关性为什么不矛盾

IV 独立性是

[
Z\perp(Y^1,Y^0,A^1,A^0),
]

意思是 (Z) 不挑人。IV 相关性是

[
Z\not\perp A,
]

意思是 (Z) 会推动人的实际处理状态。由于

[
A=ZA^1+(1-Z)A^0,
]

即使 (Z) 与潜在类型独立，也可以与观测到的 (A) 相关。

附录 C：常见卷面回答模板

C.1 为什么 Lasso 能做变量选择？

Lasso 使用 (L_1) 惩罚。几何上，(L_1) 约束区域有尖角，最优解容易落在坐标轴上，使某些系数恰好为 0。正交设计下，Lasso 解具有软阈值形式：小于阈值的 OLS 系数会被直接压到 0。因此 Lasso 同时具有系数收缩和变量选择功能。

C.2 为什么 Random Forest 比单棵树稳定？

单棵树方差大，对训练数据扰动敏感。Random Forest 通过 bootstrap 抽样训练多棵树，并在每次切分时随机选择特征子集，降低树之间相关性。多个低相关树平均或投票后，可以显著降低方差，提高预测稳定性。

C.3 为什么 accuracy 不一定可靠？

在类别不平衡时，accuracy 可能被多数类主导。例如绝大多数样本为阴性时，总是预测阴性也能获得高 accuracy，但无法发现阳性病例，sensitivity 为 0。因此分类评价应结合混淆矩阵、灵敏度、特异度、precision、recall、ROC/AUC 等指标。

C.4 什么是双重稳健？

双重稳健估计结合结果回归模型和处理机制模型。只要其中一个模型正确，目标参数估计仍然相合。它不是要求两个模型都正确，而是提供“一重正确即可”的稳健性。

C.5 为什么不能随便调整 collider？

碰撞变量形如 (A\to C\leftarrow Y)。不调整时路径被阻断；一旦条件化或调整 (C)，会打开原本封闭的路径，引入虚假关联，产生碰撞偏差。因此因果调整不能简单地“变量越多越好”。

C.6 工具变量三大条件是什么？

工具变量需要满足相关性、独立性和排他性。相关性要求 (Z) 影响处理 (A)；独立性要求 (Z) 像随机分配一样，与潜在结果和潜在处理类型独立；排他性要求 (Z) 只能通过 (A) 影响 (Y)，不能直接影响 (Y)。若还假设单调性，则 Wald 比值识别顺从者平均因果效应 CACE。

补充专题一：风险、模型选择与非参数方法细节

1. 损失、风险与经验风险

1.1 损失函数

损失函数衡量真实结果 $Y$ 与预测值 $f(X)$ 的差距。

回归中常用平方损失：

$$
L(Y,f(X))=(Y-f(X))^2.
$$

分类中常用 0-1 损失：

$$
L(f(X),Y)=
\begin{cases}
0, & f(X)=Y,\
1, & f(X)\ne Y.
\end{cases}
$$

1.2 风险

风险是总体平均损失：

$$
R(f)=E{L(Y,f(X))}.
$$

理论上最优的 $f$ 应该最小化 $R(f)$。但总体分布未知，所以不能直接最小化总体风险。

1.3 经验风险

用训练数据近似总体风险：

$$
R_n(f)=\frac1n\sum_{i=1}^n L(y_i,f(x_i)).
$$

经验风险最小化就是

$$
\hat f=\arg\min_{f\in\mathcal F}R_n(f).
$$

对平方损失，等价于

$$
\hat f=\arg\min_{f\in\mathcal F}\frac1n\sum_{i=1}^n (y_i-f(x_i))^2.
$$

2.2 最小二乘估计

残差平方和为

$$
RSS(\beta)=|y-X\beta|^2=(y-X\beta)^T(y-X\beta).
$$

普通最小二乘估计为

$$
\hat\beta^{ols}=\arg\min_\beta |y-X\beta|^2.
$$

对 $\beta$ 求导：

$$
\frac{\partial RSS}{\partial \beta}
=-2X^T(y-X\beta).
$$

令导数为零，得到正规方程：

$$
X^Ty=X^TX\beta.
$$

若 $X^TX$ 可逆，则

$$
\hat\beta^{ols}=(X^TX)^{-1}X^Ty.
$$

2.3 帽子矩阵与投影解释

拟合值为

$$
\hat y=X\hat\beta=X(X^TX)^{-1}X^Ty=Hy,
$$

其中

$$
H=X(X^TX)^{-1}X^T
$$

称为帽子矩阵。它满足：

$$
H^T=H,
\qquad
H^2=H.
$$

残差为

$$
r=y-\hat y=(I-H)y.
$$

几何解释：$\hat y$ 是 $y$ 在 $X$ 的列空间上的正交投影，残差 $r$ 与 $X$ 的列空间正交，因此

$$
X^Tr=0.
$$

2.4 线性回归估计量性质

若真实模型为

$$
Y=X^T\beta+\varepsilon,
\qquad E(\varepsilon)=0,
\qquad \operatorname{Var}(\varepsilon)=\sigma^2,
$$

则

$$
E(\hat\beta^{ols})=\beta,
$$

所以 OLS 是无偏的，并且

$$
\operatorname{Var}(\hat\beta^{ols})=\sigma^2(X^TX)^{-1}.
$$

高斯-马尔可夫定理说明：在所有无偏线性估计量中，OLS 方差最小。

但这并不意味着 OLS 预测误差总是最小。因为允许有偏估计时，可以牺牲少量偏差换取较大方差下降，这正是岭回归、LASSO 等正则化方法的动机。

3.2 模型选择的基本形式

模型选择准则通常具有形式：

$$
\text{拟合优度} + \text{复杂度惩罚}.
$$

拟合优度项随模型复杂度增加而降低，复杂度惩罚项随模型复杂度增加而升高。

常见准则：

Mallows $C_p$

$$
C_p=RSS+2\hat\sigma^2p.
$$

直观上，它用训练误差加上 $2\hat\sigma^2p$ 来近似测试误差。

AIC

$$
AIC=-2\log L+2p.
$$

BIC

$$
BIC=-2\log L+(\log n)p.
$$

当 $n$ 较大时，BIC 对变量数的惩罚比 AIC 更强，因此 BIC 往往选择更小的模型。

3.3 变量选择算法

最优子集选择

对每个模型大小 $k=1,\dots,p$，穷举所有 $k$ 个变量组合，找出 RSS 最小的模型，然后用 $C_p$、AIC、BIC 等准则选择最终模型。

优点：固定模型大小时给出全局最优。

缺点：计算量大，$p$ 大时不可行。

逐步选择

包括前向选择、后向选择、双向逐步选择。

优点：计算快。

缺点：贪心算法，只保证局部较优，不保证全局最优。

9. K 近邻分类

分类时使用多数投票：

$$
\hat C(x_0)=
\arg\max_{c\in\mathcal C}
\sum_{x_i\in N_k(x_0)} I(y_i=c).
$$

$k=1$ 时分类边界非常复杂；$k$ 增大时边界更平滑，模型复杂度下降。

14. 高维灾难

当维度 $p$ 增大时，固定样本量下样本点在空间中变得稀疏。为了找到足够多的近邻，需要覆盖很大的局部区域，导致“局部平均”不再真正局部。

例如在 $[0,1]^p$ 中，为了包含 $10/1000$ 的样本比例，超立方体边长大约为

$$
l=(10/1000)^{1/p}.
$$

当 $p$ 很大时，$l$ 会接近 1，说明所谓近邻其实已经很远。

17. 核密度估计的偏差-方差

在正则条件下，核密度估计的偏差量级约为

$$
\operatorname{Bias}(\hat f(x))=O(\lambda^2),
$$

方差量级约为

$$
\operatorname{Var}(\hat f(x))=O\left(\frac1{n\lambda}\right).
$$

因此均方误差大致为

$$
O(\lambda^4)+O\left(\frac1{n\lambda}\right).
$$

平衡两项可得最优带宽量级

$$
\lambda\asymp n^{-1/5},
$$

最优收敛率为

$$
O(n^{-4/5}).
$$

19. 核回归的边界偏差

Nadaraya-Watson 估计在定义域边界容易有偏，因为边界附近核函数支持不对称。

改进方法：局部线性回归。

对目标点 $x_0$，求解

$$
\min_{\beta_0,
\beta_1}
\sum_{i=1}^n K_\lambda(x_0,x_i)
{y_i-
\beta_0-
\beta_1x_i}^2.
$$

若写成矩阵形式，令

$$
W(x_0)=\operatorname{diag}
{K_\lambda(x_0,x_1),
\dots,K_\lambda(x_0,x_n)},
$$

则

$$
\hat\beta=(X^TWX)^{-1}X^TWy.
$$

局部线性回归可以修正一阶边界偏差。局部多项式回归可以进一步降低曲率区域的偏差，但方差会上升。

24. 贝叶斯分类规则

对多分类问题，设类别为 $1,\dots,K$。令

$$
\pi_k=P(Y=k)
$$

为先验概率，

$$
f_k(x)=p(X=x|Y=k)
$$

为第 $k$ 类下 $X$ 的条件密度。

由贝叶斯公式，

$$
P(Y=k|X=x)=
\frac{
\pi_k f_k(x)
}{
\sum_{l=1}^K \pi_l f_l(x)
}.
$$

因此贝叶斯分类规则为

$$
C^*(x)=\arg\max_k P(Y=k|X=x)

\arg\max_k \pi_k f_k(x).
$$

25.2 LDA 参数估计

先验概率估计：

$$
\hat\pi_k=
\frac{n_k}{n}.
$$

均值估计：

$$
\hat\mu_k=
\frac1{n_k}\sum_{i:y_i=k}x_i.
$$

公共协方差估计：

$$
\hat\Sigma=
\frac1{n-K}\sum_{k=1}^K\sum_{i:y_i=k}
(x_i-
\hat\mu_k)(x_i-
\hat\mu_k)^T.
$$

48. 变量重要性

随机森林常用袋外样本评估变量重要性。

对某棵树，用袋外样本计算预测误差。然后随机打乱某个变量 $X_j$ 在袋外样本中的取值，再计算预测误差。如果打乱后误差明显增加，说明该变量重要。

直观解释：变量重要性衡量“破坏该变量信息后，模型性能下降多少”。

49. 随机森林的核方法视角

随机森林也可以看作自适应核方法。

一棵树定义了一个相似度：如果训练点 $x_i$ 和目标点 $x_0$ 落在同一个终端节点，则它们相近；否则不相近。

森林把许多棵树的相似度平均，形成一个数据自适应的核。重要变量更容易被划分，因而在重要方向上带宽更小；不重要变量较少被划分，因而带宽较大。

补充专题二：高级因果推断

3.3 随机试验中 IPTW 的效率悖论

3.3.1 设定

  • (A) 随机分配，((Y^0, Y^1, L) \perp\kern-5pt\perp A)。
  • (L) 是结局 (Y) 的强预测因子。
  • 传统粗估计量（直接均值差）是有效且无偏的：
    [
    \hat{\psi}{\text{crude}} = \frac{\sum{s} \mathbf{1}(A_s=1) Y_s}{\sum_{s} \mathbf{1}(A_s=1)} - \frac{\sum_{s} \mathbf{1}(A_s=0) Y_s}{\sum_{s} \mathbf{1}(A_s=0)}
    ]

3.3.2 低效的倾向评分估计

如果统计人员忽略随机化事实，转而估计倾向评分并用IPTW，虽然仍可得到一致的估计，但方差反而更大：
[
\text{var}(\hat{\psi}{\text{crude}}) \leq \text{var}(\tilde{\psi}{\text{IPTW}})
]
原因在于：基于估计倾向评分的IPTW虽然利用了随机化机制的信息，但它以极低效的方式使用信息——它完全忽略了 (L) 是 (Y) 的强预测因子这一事实。信息量虽大，但未善加利用，反而导致效率下降。

3.3.3 启示

在随机试验中，调整强预测基线变量的正确方法是结合结局信息（如ANCOVA——协方差分析），而非仅依赖倾向评分加权。这一悖论告诫我们：并非使用了更“复杂”的方法就更好；应当根据实际数据结构选择效率最高的估计策略。

3.4 倾向评分的其他方法

3.4.1 倾向评分的关键性质：降维

若可忽略假设成立（({Y^a} \perp\kern-5pt\perp A \mid L)），则也有：
[
{Y^a} \perp\kern-5pt\perp A \mid f_{A|L}(1 \mid L)
]
即只需调整一维的倾向评分 (f_{A|L}(1|L))，就可控制整个高维向量 (L) 的混杂。这提供了降维的强大工具。

3.4.2 倾向评分回归

倾向评分回归的两阶段过程：

  1. 用 logistic 回归估计倾向评分 (\hat{f}_{A|L}(1|L; \hat{\alpha}))。
  2. 回归 (Y) 对 ({A, \hat{f}_{A|L}(1|L; \hat{\alpha})})，随后对倾向评分分布求积分或平均。

缺点：该方法通常要求同时正确指定倾向评分模型和结局模型，相比于IPTW（仅需倾向评分）、结局回归（仅需结局模型）或双重稳健估计，需要更多的建模假设。

特殊情况：加性常数效应且恒等联系
假设 (E(Y|A=1,L) - E(Y|A=0,L) = \psi) 为常数（无效应修饰），使用线性模型：
[
E[Y \mid A=a, f_{A|L}(1|L;\alpha); \eta] = \psi A + \eta_1 f_{A|L}(1|L;\alpha) + \eta_0
]
即使对 (Y) 的期望误设，OLS得到的 (\hat{\psi}) 仍然是相合的（在可忽略假设下）。因为恒等联系下，OLS的投影性质保证了 (A) 的系数一致地估计因果效应。

非线性联系时的失效
若采用 log 或 logit 等非线性联系函数，一旦模型误设，(A) 的系数不再具有因果解释。重要警示：倾向评分回归的稳健性极为有限，不可随意推广至非线性情形。

3.4.3 匹配 (Matching)

匹配方法通过为每个处理个体寻找一个或多个在倾向评分（或其他距离度量）上接近的对照个体，来消除混杂。匹配估计量的概念形式包括：

  • ATT（处理组平均处理效应）：(\widehat{ATT} = \frac{1}{N_1} \sum_{i: A_i=1} [Y_{1i} - Y_{0i}])
  • ATNT（对照组平均处理效应）：(\widehat{ATNT} = \frac{1}{N_0} \sum_{i: A_i=0} [Y_{1i} - Y_{0i}])
  • ATE：(\widehat{ATE} = \frac{1}{N} \sum_{i=1}^N [Y_{1i} - Y_{0j}])

实际操作中常用基于倾向评分的最近邻匹配或卡钳匹配。

3.6 G-估计 (G-estimation)

3.6.1 个体因果效应与秩保持模型

定义个体因果效应为：
[
\gamma_0(A, L) = Y^A - Y^0 \quad \text{以概率1成立}
]
尖锐零假设对应于 (\gamma_0(A, L) = 0)（对所有 (a)）。由一致性假设有 (Y^0 = Y - \gamma_0(A, L))。在NUCA下：
[
Y - \gamma_0(A, L) \perp\kern-5pt\perp A \mid L
]
这为G-估计提供了矩条件。

3.6.2 参数模型与估计方程

设定参数模型 (\gamma(A, L; \psi))，满足 (\gamma(A=0, L; \psi)=0)。例子：

  • 无效应修饰：(\gamma(A, L; \psi) = A\psi)
  • 线性效应修饰：(\gamma(A, L; \psi) = (A, A L’)\psi)

定义 (H(\psi) = Y - \gamma(A, L; \psi))。若倾向评分 (\pi(L) = \Pr(A=1|L)) 已知，有矩条件：
[
E[ H(\psi_0) (A - \pi(L)) m(L) ] = 0
]
其中 (m(L)) 是与 (\psi) 同维的用户选择函数。该方程称为G-估计方程。

实际估计时，通常用 logistic 回归估计 (\hat{\pi}(L))，然后解：
[
P_n[ H(\psi_g) (A - \hat{\pi}(L)) m(L) ] = 0
]
例如当模型为 (A\psi) 且 (m=1) 时：
[
\hat{\psi}_g = \frac{P_n[ Y (A - \hat{\pi}(L)) ]}{P_n[ A (A - \hat{\pi}(L)) ]}
]

3.6.3 半参数性质与效率增广

G-估计是半参数方法：无需正确指定结局分布，仅需正确指定倾向评分模型即可获得一致估计。更重要的是，在正确倾向评分模型下，可自由增广估计方程：
[
P_n[ H(\psi_g) (A - \pi(L)) m(L) ] + P_n[ t(L)(A - \pi(L)) ] = 0
]
对于任意函数 (t(L))，附加项均值为零，不影响一致性，但可用于提高效率。

3.6.4 双重稳健G-估计

最优选择 (t_{\text{opt}}(L) = -E[H(\psi_0)|L] m(L))。由NUCA和一致性，(E[H(\psi_0)|L] = E(Y|A=0, L))。对条件期望采用参数模型，得到双重稳健G-估计方程：
[
P_n[ { H(\psi_{DR}) - \hat{E}(H(\psi_{DR}) \mid L; \eta) } (A - \pi(L; \alpha)) m(L) ] = 0
]
该估计量在倾向评分模型或结局预测模型任一正确时保持一致。

3.6.5 结局回归估计的对比

直接OLS：
[
P_n( Y - \psi_{OR} A - (1, L’)\eta ) = 0
]
等价于线性模型 (Y = \psi_{OR} A + (1, L’)\eta + \epsilon)，且假定 (E(\epsilon|A,L)=0)。它完全依赖结局模型的正确设定，没有双重稳健保护。

3.6.6 秩保持模型的局限与推广

个体秩保持模型 (Y^0 = Y^A - \gamma_0(A, L)) 以概率1成立在健康科学中常不现实，因为它排除了与处理交互的未测量预测因子。幸运的是，若将 (\gamma_0) 重新定义为条件平均因果效应：
[
\gamma_0(a, l) = E(Y^a - Y^0 \mid L = l)
]
则G-估计方程仍然有效。这样 (\gamma_0) 捕捉的是在给定 (L) 子人群中干预至 (a) 与至 (0) 的反事实均值差，而不要求个体水平秩保持。这极大扩展了G-估计的适用范围。

4.2 d-分离 (d-separation)

4.2.1 定义与阻断规则

d-分离是判断在给定节点集 (B) 的条件下，节点集 (A) 与 (Y) 是否被阻断（条件独立）的形式化准则。

一条路径在给定 (B) 下被阻断，当且仅当满足以下任一条件：

  1. 存在一个非对撞节点在 (B) 中：链状 (X \rightarrow M \rightarrow Y) 或叉状 (X \leftarrow M \rightarrow Y)，当条件化 (M) 时，(X) 与 (Y) 之间的路径被阻断。
  2. 存在一个对撞节点不在 (B) 中，且该对撞节点的所有后代也不在 (B) 中：对撞节点本身已阻断路径；但若条件化了对撞节点或其某个后代，反而会打开这条路径（产生对撞偏倚）。

如果 (A) 与 (Y) 之间的每一条路径都被 (B) 阻断，则称 (A) 与 (Y) 被 (B) d-分离。若存在至少一条未阻断路径，则称 (A) 与 (Y) d-连接。

4.2.2 对撞节点的特殊性：数值示例

考虑幻灯片中的示例：A 与 B 是两个独立的原因，C 是他们的共同效应（A→C←B）。数据满足边缘独立（Pr(A=1|B)=Pr(A=1)=0.5），但在给定C后，A与B产生关联（条件优势比不为1）。这证明条件化碰撞节点会打开原本阻断的路径。

这种对撞偏倚（也叫 Berkson 偏倚或选择偏倚）在实际中极易发生：例如，只选择住院患者进行分析，可能因为“是否住院”是疾病和暴露的共同结果而产生虚假关联。

4.2.3 条件集的选择

d-分离强调：条件化正确节点可隔绝混杂，条件化错误节点（如对撞节点或中介变量）反而引入偏倚。幻灯片76页展示了给定空集 (\emptyset) 和给定 ({Q}) 时路径状态的变化——原本d-分离的路径在条件化对撞节点 (Q) 后变为d-连接。这提醒我们盲目条件化可能引入选择偏倚。

4.3 后门路径准则 (Backdoor Path Criterion)

4.3.1 定义与操作

后门路径准则提供了一个基于DAG的图形化方法，用于识别哪些变量集足以控制混杂。

正式陈述：

一个变量集 (S) 是充分的调整集，当且仅当：

  1. (S) 中不包含暴露 (E) 的任何后代；
  2. 在删除所有从 (E) 发出的箭头后的图中，(S) d-分离 (E) 与 (D)。

条件(1)确保不会条件化受暴露影响的中介变量或碰撞路径上的节点；条件(2)确保所有后门路径均被阻断，仅保留因果路径。

4.3.2 哮喘示例分析

在图1（哮喘DAG）中：

  • 暴露E，结局D。
  • 后门准则识别出充分调整集 (S = {A, C}) 或 (S = {B, C})。调整这些变量即可获得E对D的无偏因果效应。
  • 仅调整 (C) 是不够的：虽然 (C) 阻断了大多数后门路径，但 (C) 是 (A) 和 (B) 的共同效应（碰撞节点），条件化 (C) 会打开 (A) 与 (B) 之间的路径，从而引入新的混杂路径 (E \leftarrow A - B \rightarrow D)（其中 (A) 与 (B) 在条件化 (C) 后产生关联）。因此必须同时调整 (A) 和 (C)，或 (B) 和 (C)，才能完全阻断所有后门路径。

4.3.3 充分调整集的非唯一性

充分调整集可能有多个，选择时应考虑变量的测量质量和可获取性。后门准则相较于传统“与暴露和结局都相关”的经验判断，提供了严谨的图形化充分条件，避免了调整对撞节点或中介变量的常见错误。

4. K-means + Mahalanobis Distance

2024 回忆卷中说“用 k-means 方法简单分类，使用马氏距离，(k=3)，具体数字记不清”。这里给出通用做题模板。

4.1 普通 K-means 算法

K-means 的目标是把样本分成 (K) 类，使类内平方距离和最小：

[
\min_{C_1,\ldots,C_K}
\sum_{k=1}^K
\sum_{x_i\in C_k}
|x_i-\mu_k|^2,
]

其中

[
\mu_k=\frac{1}{|C_k|}\sum_{x_i\in C_k}x_i
]

是第 (k) 类的中心。

算法步骤：

  1. 初始化 (K) 个中心；
  2. 对每个样本 (x_i)，计算它到每个中心的距离；
  3. 把 (x_i) 分到最近的中心；
  4. 根据新分组重新计算每个中心；
  5. 重复直到分类不再改变或目标函数收敛。

4.2 Mahalanobis Distance

普通欧氏距离是

[
d_E(x,\mu)=\sqrt{(x-\mu)^T(x-\mu)}.
]

马氏距离是

[
d_M(x,\mu)

\sqrt{(x-\mu)^T\Sigma^{-1}(x-\mu)}.
]

其中 (\Sigma) 是协方差矩阵。

如果只比较大小，可以不取平方根，直接比较

[
d_M^2(x,\mu)

(x-\mu)^T\Sigma^{-1}(x-\mu).
]

4.3 考试做题模板

如果给出一个样本点 (x) 和三个中心 (\mu_1,\mu_2,\mu_3)，并给出协方差矩阵 (\Sigma)，则计算：

[
d_1^2=(x-\mu_1)^T\Sigma^{-1}(x-\mu_1),
]

[
d_2^2=(x-\mu_2)^T\Sigma^{-1}(x-\mu_2),
]

[
d_3^2=(x-\mu_3)^T\Sigma^{-1}(x-\mu_3).
]

然后选择距离最小的类：

[
\hat C(x)=\arg\min_{k=1,2,3} d_k^2.
]

5. 判断题：树和随机森林

5(a). 若 (X_1) 与 (Y) 独立，则 (X_1) 不会出现在树结构中？

这个说法一般应判断为：

[
\boxed{\text{错误或不严谨。}}
]

原因有两个层面。

第一，有限样本下

即使总体上 (X_1) 与 (Y) 独立，在有限训练样本中也可能出现随机相关。决策树是贪心算法，它会根据训练样本中的 impurity reduction 选择切分变量，因此 (X_1) 仍然可能被选中。

第二，边际独立不代表没有交互作用

可能出现

[
X_1 \perp Y,
]

但 (X_1) 与其他变量一起对 (Y) 有作用。典型例子是 XOR 结构：

[
Y=X_1\oplus X_2.
]

单独看 (X_1) 与 (Y) 可能独立，但在给定 (X_2) 后，(X_1) 对 (Y) 有预测作用。因此树中仍可能使用 (X_1)。

考试模板

该说法不一定正确。决策树根据样本中的 impurity reduction 进行贪心切分；即使某变量与 (Y) 边际独立，有限样本随机波动或变量交互都可能使其出现在树结构中。

5(b). 树的最大深度小于 (\log_2 n+1)？

一般判断：

[
\boxed{\text{错误。}}
]

如果一棵二叉树完全平衡，则深度大约为

[
O(\log_2 n).
]

但未经剪枝的决策树不一定平衡。极端情况下，每次切分只分出一个样本，树可以非常不平衡，深度可以达到

[
O(n).
]

所以最大深度不一定小于 (\log_2 n+1)。

考试模板

(\log_2 n) 量级只对应平衡二叉树。未剪枝决策树可能高度不平衡，最大深度可达到 (n) 量级，因此该说法错误。

5(c). 随机森林是树的求平均？

这个说法要分任务。

回归问题

随机森林的预测通常是多棵树预测值的平均：

[
\hat f_{RF}(x)

\frac1B\sum_{b=1}^B \hat f_b(x).
]

所以在回归中，“随机森林是树的平均”可以接受。

分类问题

分类时，随机森林通常采用多数投票：

[
\hat y_{RF}(x)

\operatorname{majority vote}
{\hat y_1(x),\ldots,\hat y_B(x)}.
]

也可以平均各类概率再分类，但不一定直接说“求平均”。

答案

如果题目没有限定回归：

[
\boxed{\text{不完全正确。}}
]

考试模板：

随机森林是多棵决策树的集成。回归任务中通常对树的预测值取平均，分类任务中通常采用多数投票。因此笼统说“随机森林是树的求平均”不够严谨。

5(d). 树可以用于预测和分类，随机森林只能用于预测，不能用于分类？

判断：

[
\boxed{\text{错误。}}
]

决策树可以用于：

  • 回归；
  • 分类。

随机森林也可以用于：

  • 回归；
  • 分类。

分类随机森林通过多数投票或类别概率平均给出分类结果。

1. LASSO 估计的性质

1(1). 正交设计下推导 LASSO 估计

题目给出的结论是

[
\hat\beta_j^{lasso}

\begin{cases}
\hat\beta_j^{ols}-\lambda/2, & \hat\beta_j^{ols}>\lambda/2,\
0, & |\hat\beta_j^{ols}|\leq \lambda/2,\
\hat\beta_j^{ols}+\lambda/2, & \hat\beta_j^{ols}<-\lambda/2.
\end{cases}
]

这就是 soft-thresholding 形式。

Step 1：写出 LASSO 目标函数

假设 LASSO 使用的目标函数是

[
Q(\beta)

|Y-X\beta|^2+\lambda\sum_{j=1}^p|\beta_j|.
]

如果 (X) 的列正交，并且为了得到题目中的 (\lambda/2) 阈值，我们可以取标准化情形

[
X^TX=I.
]

OLS 估计为

[
\hat\beta^{ols}=X^TY.
]

Step 2：把目标函数按坐标分解

有

[
|Y-X\beta|^2

Y^TY-2\beta^TX^TY+\beta^TX^TX\beta.
]

因为

[
X^TX=I,
]

所以

[
|Y-X\beta|^2

Y^TY-2\sum_{j=1}^p\beta_j\hat\beta_j^{ols}
+\sum_{j=1}^p\beta_j^2.
]

对每个 (j) 来说，和 (\beta_j) 有关的部分是

[
\beta_j^2-2\beta_j\hat\beta_j^{ols}
+\lambda|\beta_j|.
]

配方：

[
\beta_j^2-2\beta_j\hat\beta_j^{ols}

(\beta_j-\hat\beta_j^{ols})^2

(\hat\beta_j^{ols})^2.
]

其中

[
-(\hat\beta_j^{ols})^2
]

与 (\beta_j) 无关，可以忽略。

于是每个坐标的问题化为

[
\min_{\beta_j}
\left{
(\beta_j-\hat\beta_j^{ols})^2+\lambda|\beta_j|
\right}.
]

Step 3：分三种情况讨论

情况一：(\beta_j>0)

此时

[
|\beta_j|=\beta_j.
]

目标函数为

[
(\beta_j-\hat\beta_j^{ols})^2+\lambda\beta_j.
]

求导：

[
2(\beta_j-\hat\beta_j^{ols})+\lambda=0.
]

所以

[
\beta_j

\hat\beta_j^{ols}-\frac{\lambda}{2}.
]

但这个解属于当前情况，需要满足

[
\beta_j>0.
]

即

[
\hat\beta_j^{ols}-\frac{\lambda}{2}>0.
]

因此

[
\hat\beta_j^{ols}>\frac{\lambda}{2}.
]

所以若

[
\hat\beta_j^{ols}>\frac{\lambda}{2},
]

则

[
\hat\beta_j^{lasso}

\hat\beta_j^{ols}-\frac{\lambda}{2}.
]

情况二：(\beta_j<0)

此时

[
|\beta_j|=-\beta_j.
]

目标函数为

[
(\beta_j-\hat\beta_j^{ols})^2-\lambda\beta_j.
]

求导：

[
2(\beta_j-\hat\beta_j^{ols})-\lambda=0.
]

所以

[
\beta_j

\hat\beta_j^{ols}+\frac{\lambda}{2}.
]

这个解属于当前情况，需要满足

[
\beta_j<0.
]

即

[
\hat\beta_j^{ols}+\frac{\lambda}{2}<0.
]

因此

[
\hat\beta_j^{ols}<-\frac{\lambda}{2}.
]

所以若

[
\hat\beta_j^{ols}<-\frac{\lambda}{2},
]

则

[
\hat\beta_j^{lasso}

\hat\beta_j^{ols}+\frac{\lambda}{2}.
]

情况三：(\beta_j=0)

因为 (|\beta_j|) 在 0 处不可导，所以使用次梯度。

目标函数为

[
q(\beta_j)

(\beta_j-\hat\beta_j^{ols})^2+\lambda|\beta_j|.
]

在 (\beta_j=0) 处，第一项的导数是

[
2(0-\hat\beta_j^{ols})

-2\hat\beta_j^{ols}.
]

而

[
\partial |\beta_j| \big|_{\beta_j=0}

[-1,1].
]

所以次梯度条件为

[
0\in -2\hat\beta_j^{ols}+\lambda[-1,1].
]

这等价于存在 (s\in[-1,1])，使得

[
0=-2\hat\beta_j^{ols}+\lambda s.
]

即

[
2\hat\beta_j^{ols}=\lambda s.
]

因为

[
s\in[-1,1],
]

所以

[
2\hat\beta_j^{ols}\in[-\lambda,\lambda].
]

等价于

[
|\hat\beta_j^{ols}|\leq \frac{\lambda}{2}.
]

因此若

[
|\hat\beta_j^{ols}|\leq \frac{\lambda}{2},
]

则

[
\hat\beta_j^{lasso}=0.
]

最终结论

[
\boxed{
\hat\beta_j^{lasso}

\begin{cases}
\hat\beta_j^{ols}-\lambda/2, & \hat\beta_j^{ols}>\lambda/2,\
0, & |\hat\beta_j^{ols}|\leq \lambda/2,\
\hat\beta_j^{ols}+\lambda/2, & \hat\beta_j^{ols}<-\lambda/2.
\end{cases}
}
]

也可以写成紧凑形式：

[
\hat\beta_j^{lasso}

\operatorname{sign}(\hat\beta_j^{ols})
\left(
|\hat\beta_j^{ols}|-\frac{\lambda}{2}
\right)_+.
]

1(2). LASSO 估计是否有偏？

答案：

[
\boxed{\text{一般有偏。}}
]

原因是 LASSO 会把 OLS 估计往 0 方向收缩。

例如当

[
\hat\beta_j^{ols}>\frac{\lambda}{2}
]

时，

[
\hat\beta_j^{lasso}

\hat\beta_j^{ols}-\frac{\lambda}{2}.
]

即使 (\hat\beta_j^{ols}) 是无偏的，LASSO 也系统性地减小其绝对值。

所以通常

[
E(\hat\beta_j^{lasso})\neq \beta_j.
]

考试模板

LASSO 一般是有偏估计。因为 (L_1) 正则化会将估计量向 0 收缩，并可能把部分系数压为 0。它牺牲一部分 bias 来换取更小的 variance 和变量选择能力。

6. 生存分析：Cox 比例风险模型

题目给定 Cox 模型：

[
h(t\mid A)=h_0(t)\exp(\beta A).
]

数据：

个体 事件时间 删失指标 协变量 (A)
1 2.0 1 1
2 3.5 1 0
3 3.0 1 1
4 4.5 0 0

这里删失指标为 1 表示发生事件，0 表示删失。

6(1). 写出偏似然表达式，并解释 (\beta) 的含义

Cox 模型为

[
h(t\mid A)=h_0(t)\exp(\beta A).
]

对于 (A=1) 和 (A=0)，风险函数比值为

[
\frac{h(t\mid A=1)}{h(t\mid A=0)}

\frac{h_0(t)\exp(\beta)}
{h_0(t)}

\exp(\beta).
]

所以

[
\boxed{\exp(\beta)\text{ 是 }A=1\text{ 相对于 }A=0\text{ 的 hazard ratio。}}
]

若

[
\beta>0,
]

则

[
\exp(\beta)>1,
]

说明 (A=1) 增加事件发生风险。

若

[
\beta<0,
]

则

[
\exp(\beta)<1,
]

说明 (A=1) 降低事件发生风险。

Cox 偏似然公式

对每一个发生事件的时间 (t_i)，构造风险集

[
R(t_i)={j:T_j\geq t_i}.
]

偏似然为

[
L(\beta)

\prod_{i:\delta_i=1}
\frac{\exp(\beta A_i)}
{\sum_{j\in R(t_i)}\exp(\beta A_j)}.
]

Step 1：找出事件时间

发生事件的个体是：

  • 个体 1：时间 2.0，(A=1)；
  • 个体 3：时间 3.0，(A=1)；
  • 个体 2：时间 3.5，(A=0)。

个体 4 在 4.5 被删失，不作为分子中的事件者，但在它删失之前属于风险集。

事件时间排序：

[
2.0,\quad 3.0,\quad 3.5.
]

Step 2：写风险集

时间 (t=2.0)

此时所有人都还在风险集中：

[
R(2.0)={1,2,3,4}.
]

对应 (A) 值为：

[
1,0,1,0.
]

分母为

[
\exp(\beta)+1+\exp(\beta)+1

2\exp(\beta)+2.
]

事件者为个体 1，(A=1)，分子为

[
\exp(\beta).
]

所以第一项是

[
\frac{\exp(\beta)}{2\exp(\beta)+2}.
]

时间 (t=3.0)

此时还在风险集中的人是时间至少为 3.0 的个体：

[
R(3.0)={2,3,4}.
]

对应 (A) 值为：

[
0,1,0.
]

分母为

[
1+\exp(\beta)+1

\exp(\beta)+2.
]

事件者为个体 3，(A=1)，分子为

[
\exp(\beta).
]

所以第二项是

[
\frac{\exp(\beta)}{\exp(\beta)+2}.
]

时间 (t=3.5)

此时还在风险集中的人是：

[
R(3.5)={2,4}.
]

对应 (A) 值为：

[
0,0.
]

分母为

[
1+1=2.
]

事件者为个体 2，(A=0)，分子为

[
\exp(0)=1.
]

所以第三项是

[
\frac12.
]

最终偏似然

因此

[
L(\beta)

\frac{\exp(\beta)}{2\exp(\beta)+2}
\cdot
\frac{\exp(\beta)}{\exp(\beta)+2}
\cdot
\frac12.

7. 因果图：是否需要调整 (X)

题目问：检测 (A) 对 (Y) 的因果效应时，哪个图需要控制 (X)，哪个不需要？

图 1：(A\leftarrow X\rightarrow Y)

这是

[
A\leftarrow X\rightarrow Y.
]

这里 (X) 是 (A) 和 (Y) 的共同原因，即 confounder。

路径

[
A\leftarrow X\rightarrow Y
]

是 backdoor path。若不控制 (X)，会产生混杂偏差。

因此需要控制 (X)。

答案：

[
\boxed{\text{需要控制 }X.}
]

图 2：(A\rightarrow X\leftarrow Y)

这是

[
A\rightarrow X\leftarrow Y.
]

这里 (X) 是 collider，碰撞点。

在不控制 (X) 时，路径

[
A\rightarrow X\leftarrow Y
]

被 collider 阻断。

如果控制 (X)，反而会打开这条路径，引入 collider bias。

因此不能控制 (X)。

答案：

[
\boxed{\text{不需要，且不应该控制 }X.}
]

图 3：(A\rightarrow X\rightarrow Y)

这是

[
A\rightarrow X\rightarrow Y.
]

这里 (X) 是 mediator，中介变量。

如果目标是估计 (A) 对 (Y) 的总效应，那么路径

[
A\rightarrow X\rightarrow Y
]

本身是因果效应的一部分。

如果控制 (X)，就会阻断部分因果效应，只能估计 direct effect，而不是 total effect。

因此在估计总效应时，不应控制 (X)。

答案：

[
\boxed{\text{估计总效应时不控制 }X.}
]

总结表

图 类型 是否控制 (X)
(A\leftarrow X\rightarrow Y) 混杂变量 控制
(A\rightarrow X\leftarrow Y) 碰撞变量 不控制
(A\rightarrow X\rightarrow Y) 中介变量 估计总效应时不控制

9. 工具变量

本题对应 2025 春夏卷的工具变量部分。下面仍采用工具变量课件中的反事实记号：(A^z) 表示把工具变量设为 (z) 时的潜在处理状态，(Y^a) 表示把处理设为 (a) 时的潜在结局。

9(1). 以下哪种情况违反排他性约束？

选项：

A. (Z) 与 (A) 弱相关
B. 存在未观测混杂，使得 (Z) 通过该混杂影响 (Y)
C. (Z) 只对 treated group 有影响
D. (Z) 与 (Y) 的随机误差项 (\varepsilon) 相关

先写清楚排他性约束是什么

排他性约束要求：工具变量 (Z) 只能通过处理 (A) 影响结局 (Y)，不能直接影响 (Y)，也不能通过其他不经过 (A) 的路径影响 (Y)。用反事实形式写为

[
Y_{z,a}=Y_a.
]

它的含义是：固定处理状态 (A=a) 后，改变 (Z) 不应再改变结局。

逐项分析

A 说 (Z) 与 (A) 弱相关。这个主要违反的是相关性假设。若 (Z) 几乎不影响 (A)，Wald 比值的分母会很小，估计不稳定；若完全不相关，则工具变量无效。但这不是排他性约束本身。

B 说存在未观测混杂，使得 (Z) 通过该混杂影响 (Y)。这会破坏工具变量的独立性或无共同原因假设。严格按照课件的假设分类，它主要对应

[
Z\not\perp (Y^1,Y^0,A^1,A^0).
]

也就是说，(Z) 不再像随机分配一样。不过如果题目把这句话理解为 (Z) 存在一条不经过 (A) 到达 (Y) 的路径，那么它也会使工具变量失效。

C 说 (Z) 只对 treated group 有影响。这句话本身不够精确。如果意思是 (Z) 只影响接受处理的概率 (A)，则不一定违反排他性；如果意思是即使固定 (A=1)，(Z) 仍然直接影响 (Y)，即

[
Y_{1,1}\neq Y_{0,1},
]

那就违反排他性约束。

D 说 (Z) 与 (Y) 的随机误差项 (\varepsilon) 相关。在结构方程语言中，误差项 (\varepsilon) 表示结局中不由处理 (A) 解释的部分。如果 (Z) 与 (\varepsilon) 相关，说明 (Z) 与 (Y) 中不经过 (A) 的部分有关，因此通常判为违反排他性 / 外生性条件。

所以如果考试是单选，最稳答案是

[
\boxed{D}.
]

若考试允许说明，可以写：A 主要违反相关性；B 主要违反独立性；D 明确说明 (Z) 与结局中不经由 (A) 的部分相关，因此最符合“排他性约束被破坏”的表述。C 要看具体含义，若表示固定 (A) 后 (Z) 仍影响 (Y)，则也违反排他性。

9(2). 四类人群的定义；观察到 (Z=1,A=0) 属于哪类？

定义潜在处理状态：

[
A^1=\text{如果 }Z=1\text{ 时的处理状态},
]

[
A^0=\text{如果 }Z=0\text{ 时的处理状态}.
]

根据 ((A^0,A^1))，人群分为四类：

类型 定义 含义
永不接受者 (A^0=0,A^1=0) 无论 (Z=0) 还是 (Z=1)，都不接受处理
总是接受者 (A^0=1,A^1=1) 无论 (Z=0) 还是 (Z=1)，都接受处理
服从者 (A^0=0,A^1=1) (Z=1) 时接受处理，(Z=0) 时不接受处理
违抗者 (A^0=1,A^1=0) (Z=1) 时不接受处理，(Z=0) 时反而接受处理

如果观察到

[
Z=1,\quad A=0,
]

由一致性 (A=A^Z) 可得

[
A^1=0.
]

因此该个体可能是：

[
(A^0,A^1)=(0,0),
]

即永不接受者；也可能是

[
(A^0,A^1)=(1,0),
]

即违抗者。

所以如果没有单调性假设，答案是

[
\boxed{\text{可能是永不接受者，也可能是违抗者。}}
]

若再假设单调性

[
A^1\ge A^0,
]

则违抗者被排除，因此观察到 (Z=1,A=0) 的人只能是永不接受者。

9(3). 若想识别总是接受者群体中的平均因果效应，需要附加什么条件？

总是接受者定义为

[
A^0=1,\quad A^1=1.
]

它们无论 (Z=0) 还是 (Z=1)，都会接受处理。总是接受者群体中的平均因果效应是

[
E(Y^1-Y^0\mid A^0=1,A^1=1).
]

问题在于：总是接受者总是接受处理，所以在观测数据中通常只能看到它们接受处理时的结局 (Y^1)，看不到它们不接受处理时的潜在结局 (Y^0)。因此，仅靠标准工具变量假设，一般只能识别服从者平均因果效应，而不能直接识别总是接受者的平均因果效应。

一个足够的附加条件：处理效应在不同人群中相同

可以附加处理效应同质性假设，即不同服从类型的人群具有相同的平均处理效应。例如假设

[
E(Y^1-Y^0\mid A^0=1,A^1=1)

E(Y^1-Y^0\mid A^0=0,A^1=1).
]

右边是服从者平均因果效应，也就是 CACE。由于标准工具变量假设可以通过 Wald 比值识别 CACE，所以在上述同质性假设下，总是接受者的平均因果效应也可以被识别，并且等于 CACE：

[
E(Y^1-Y^0\mid A^0=1,A^1=1)=CACE.
]

更强的写法是要求三类人群的平均处理效应相同：

[
E(Y^1-Y^0\mid A^0=0,A^1=1)

E(Y^1-Y^0\mid A^0=0,A^1=0)

E(Y^1-Y^0\mid A^0=1,A^1=1).
]

在这种假设下，Wald 比值不仅可以解释为 CACE，也可以解释为总体平均处理效应或总是接受者的平均处理效应。

考试可背版

标准工具变量假设通常只能识别服从者平均因果效应
[
CACE=E(Y^1-Y^0\mid A^0=0,A^1=1),
]
不能直接识别总是接受者的平均因果效应。若额外假设总是接受者和服从者的平均处理效应相同，即
[
E(Y^1-Y^0\mid A^0=1,A^1=1)

E(Y^1-Y^0\mid A^0=0,A^1=1),
]
则总是接受者的平均因果效应等于 CACE，因此可由 Wald 比值识别。

9.4 工具变量部分总模板

考试中如果问“工具变量能识别什么”，最稳写法是：

在相关性、独立性、排他性、一致性、正值性和单调性假设下，工具变量的 Wald 比值
[
\frac{E(Y\mid Z=1)-E(Y\mid Z=0)}
{E(A\mid Z=1)-E(A\mid Z=0)}
]
识别的是服从者平均因果效应
[
E(Y^1-Y^0\mid A^0=0,A^1=1),
]
而不是一般情况下的总体平均处理效应。若进一步加入处理效应同质性假设，则可以把该比值解释为更广泛人群的平均处理效应。
